# ADR-003：投资组合框架（2/3 白马 + 1/3 弹性）

- **状态**：已采纳
- **日期**：2026-05-13
- **决策者**：Daniel Wong

---

## 背景

当前 A 股 AI 基建（光模块/存储/芯片/PCB 等）处于多年难遇的窗口期：
- 2025-2026 AI Agent 推理端爆发，算力需求不可逆
- BIS 出口管制持续升级，国产替代加速
- 超大云厂商资本支出 2026 年合计 $3,150 亿（+48% YoY）

已有的研报（aidc_report.html）只覆盖细分行业龙头（白马股为主）。
需要引入**弹性股**以最大化当前窗口期收益，同时控制回撤风险。

核心问题：**如何在确定性和高弹性之间科学分配仓位？**

---

## 框架定义

### 白马股（2/3，~67%）

**特征：**
- 细分行业龙头，市场地位不可撼动
- 紧缺度 Tier-1 或 Tier-2
- 流动性好（日均成交 >1 亿）
- 机构重仓，外资北向青睐
- 有清晰的分析师目标价覆盖

**代表标的：**

| 标的 | 代码 | 理由 | 稀缺度 |
|---|---|---|---|
| 中际旭创 | 300308 | 全球光模块#1，Q1营收+192%，1.6T 放量 | Tier-1 |
| 澜起科技 | 688008 | 全球内存接口芯片#1，RCD5 供不应求 | Tier-1 |
| 北方华创 | 002371 | 国产半导体设备#1，BIS 管制受益最大 | Tier-2 |
| 深南电路 | 002916 | 国内 AI 高速 PCB 最强，高层数 PCB 壁垒 | Tier-2 |
| 英维克 | 002837 | 国内液冷#1，冷板式液冷 30%+渗透率趋势 | Tier-1 |
| 浪潮信息 | 000977 | AI 服务器国内出货#1，估值合理 | Tier-2 |

### 弹性股（1/3，~33%）

**特征：**
- 赛道早期或转型中，高风险高回报
- 小市值（<500 亿），股价弹性大
- 依赖单一产品突破或政策催化
- 波动率显著高于白马股（Beta > 1.5）
- 仓位单只不超过总仓位 8%

**代表标的：**

| 标的 | 代码 | 弹性来源 | 风险点 |
|---|---|---|---|
| 天孚通信 | 300394 | CPO（共封装光学）量产突破 | CPO 商业化进度不及预期 |
| 星融元 | 688616 | 国产 RoCE 以太网交换机 | 营收规模尚小，亏损期 |
| 剑桥科技 | 603083 | OCS 光交换新赛道 | 新业务占比低，主营转型风险 |
| 高澜股份 | 300499 | 小市值冷源系统，AIDC 冷源需求爆发 | 市值小、流动性差 |
| 寒武纪 | 688256 | AI 芯片国产替代唯一量产平台 | 持续亏损，估值严重透支 |

## 可选框架对比

| 框架 | Pros | Cons | 适用场景 |
|---|---|---|---|
| **2/3 白马 + 1/3 弹性（选定）** | 有纪律；风险可控；牛市有弹性；回调有防御 | 需要周期性再平衡 | 当前窗口期，有入场机会但需控制回撤 |
| 全白马（100%） | 最稳健；回撤最小 | 牛市相对收益跑输；错过弹性股行情 | 熊市/震荡市 |
| 全弹性（100%） | 牛市弹性最大 | 回撤不可控（-50% 以上可能）；精神压力大 | 不建议 |
| 动态再平衡（纯信号驱动） | 理论最优 | 对模型质量强依赖；小股票池易过拟合 | 有足够历史数据后考虑 |
| 核心卫星（70%+30%） | 成熟机构策略 | 与当前策略类似，命名不同 | 等价替代 |

---

## 动态调整规则

### 触发器设计

| 触发条件 | 调整方向 | 幅度 |
|---|---|---|
| 超大云厂商 CapEx 指引上调（季度） | 弹性股 ↑ | +7% → 最高 40% |
| 超大云厂商 CapEx 指引下调 >10% | 弹性股 ↓ | -13% → 最低 20% |
| 板块整体回调 >15%（且基本面无变化） | 白马股 ↑ | +7% → 最高 75% |
| 板块突破前高，量价齐升 | 弹性股 ↑ | +5% → 最高 38% |
| 单只弹性股亏损 >20% | 触发止损提示 | 建议减仓或清仓 |
| 单只标的达到目标价 80% | 触发减仓提示 | 建议减仓 30-50% |
| 弹性股单只市值占比超过总仓位 10% | 再平衡提示 | 建议减至 8% 以下 |

### 宏观门控层（CapEx 周期分类器）

```
增长状态（Green）：四大云厂商 CapEx 指引 QoQ ≥ +5%
  → 弹性股目标权重上调到 38-40%

平稳状态（Yellow）：CapEx 指引 -5% ~ +5%
  → 维持 2/3+1/3 基准框架

收缩状态（Red）：CapEx 指引 QoQ < -10%
  → 弹性股目标权重降至 20%，白马股提升至 80%
```

## 仓位校验算法（系统实现）

```python
def check_portfolio_balance(holdings_df, macro_state="green"):
    target_elastic = {"green": 0.38, "yellow": 0.33, "red": 0.20}[macro_state]
    target_white = 1 - target_elastic

    current_elastic = holdings_df[holdings_df["category"] == "弹性股"]["market_value"].sum()
    total = holdings_df["market_value"].sum()
    current_elastic_ratio = current_elastic / total

    deviation = current_elastic_ratio - target_elastic
    if abs(deviation) > 0.10:  # 偏差超过 10% 才提示
        return {
            "rebalance_needed": True,
            "direction": "reduce_elastic" if deviation > 0 else "add_elastic",
            "deviation": deviation
        }
    return {"rebalance_needed": False, "deviation": deviation}
```

## Trade-off 推导

**为什么选 2/3+1/3 而不是动态再平衡？**

动态再平衡（纯信号驱动）在理论上最优，但有两个关键障碍：
1. **小股票池过拟合**：50-100 只 AI 基建股的历史数据不足以训练稳定的 ML 仓位模型
2. **本工具定位是辅助建议而非自动执行**：过于复杂的动态模型反而降低建议的可解释性

2/3+1/3 框架的优势：
- 每个持仓决策都能用人话解释（这是白马还是弹性，仓位超配了没有）
- 宏观门控（CapEx 周期）作为粗粒度调整，不需要高频计算
- 投资纪律强制执行：弹性股不允许超过 40%，杜绝"追热点全仓"

---

## 后果

- ✅ 窗口期最大化收益：弹性股捕捉 CPO/RoCE/OCS 等新赛道爆发
- ✅ 回撤可控：2/3 白马股在板块回调时提供缓冲
- ✅ 建议可解释：每条建议都能说清楚是"白马加仓"还是"弹性止损"
- ⚠️ 需要每日检查仓位比例（系统自动计算，提示人工操作）
- ⚠️ 弹性股止损纪律必须严格执行（-20% 是硬线，不能带止损）
