## `engine/backtest/runner.py`

```python
from dataclasses import dataclass
import pandas as pd

@dataclass
class BacktestResult:
    code: str
    strategy: str
    total_return: float       # 累计收益率（小数，如 0.25 = 25%）
    benchmark_return: float   # 买入持有收益率
    max_drawdown: float       # 最大回撤（负数，如 -0.15）
    win_rate: float           # 胜率
    num_trades: int
    equity_curve: pd.Series   # index=DatetimeIndex, values=累计净值（起始=1.0）

def run_backtest(
    code: str,
    strategy: str,    # "ma_cross" | "macd_cross"
    df: pd.DataFrame, # OHLCV DataFrame，必须含 close 列（小写）
) -> BacktestResult: ...
```

### 指标计算口径

| 指标 | 计算方式 |
|------|---------|
| `equity_curve` | 初始净值 1.0；持仓期间每日乘以 `(close_t / close_{t-1})`；空仓期净值不变 |
| `total_return` | `equity_curve.iloc[-1] - 1.0` |
| `benchmark_return` | `close.iloc[-1] / close.iloc[0] - 1.0` |
| `max_drawdown` | `(equity_curve / equity_curve.cummax() - 1.0).min()`；无交易时为 `0.0` |
| `win_rate` | 盈利交易数 / `num_trades`；`num_trades == 0` 时为 `0.0` |
| `num_trades` | signal 从 0→1 的次数（买入次数） |

一笔交易：买入（signal 0→1）至卖出（signal 1→0）；末尾如仍持仓，以最后收盘价平仓。

### 边界条件与异常

| 情况 | 行为 |
|------|------|
| 无交易信号（signal 全 0） | 返回 `BacktestResult(total_return=0.0, max_drawdown=0.0, win_rate=0.0, num_trades=0, equity_curve=Series(1.0))` |
| `df` 为空 DataFrame | `raise ValueError("Empty OHLCV data")` |
| 缺少 `close` 列 | `raise KeyError("close column required")` |
| `close` 含 NaN | `raise ValueError("NaN values in close column")` |
| 数据行数不足（< 26 行，不足以计算 MA20 或 MACD） | `raise ValueError("Insufficient data: need ≥ 26 rows")` |
| `strategy` 非 `ma_cross` / `macd_cross` | `raise ValueError(f"Unknown strategy: {strategy!r}")` |
| index 不是 DatetimeIndex 或未升序 | 函数内部 `sort_index()` 后处理，不报错 |

## `engine/backtest/strategies.py`

两个策略函数，接收 DataFrame（需含 `close` 列），返回 signal Series（1=持有, 0=空仓）：

- `ma_cross_signals(df) -> pd.Series`：MA5 上穿 MA20 买入，下穿卖出；`signal.shift(1)` 避免 lookahead bias
- `macd_cross_signals(df) -> pd.Series`：DIF（EMA12−EMA26）上穿 DEA（DIF 的 EMA9）买入，下穿卖出；`signal.shift(1)` 避免 lookahead bias
