# Goal

基于历史 OHLCV 数据对 MA 均线策略 + MACD 策略进行回测，量化评估当前持仓标的的信号质量，以单独 PR 发布。

# In Scope

## `engine/backtest/__init__.py`

空文件，建立模块。

## `engine/backtest/runner.py`

```python
from dataclasses import dataclass
import pandas as pd

@dataclass
class BacktestResult:
    code: str
    strategy: str
    total_return: float       # 累计收益率（小数，如 0.25 = 25%）
    benchmark_return: float   # 买入持有收益率
    max_drawdown: float       # 最大回撤（负数，如 -0.15 = 15% 回撤）
    win_rate: float           # 胜率
    num_trades: int
    equity_curve: pd.Series   # index=date, values=累计净值

def run_backtest(
    code: str,
    strategy: str,    # "ma_cross" | "macd_cross"
    df: pd.DataFrame, # OHLCV DataFrame（Close 列必须存在）
) -> BacktestResult: ...
```

## `engine/backtest/strategies.py`

两个策略函数，接收 DataFrame，返回 signal Series（1=持有, 0=空仓）：
- `ma_cross_signals(df)`: MA5 上穿 MA20 买入，下穿卖出
- `macd_cross_signals(df)`: DIF 上穿 DEA 买入，下穿卖出

## `scripts/run_backtest.py`

CLI 入口：
```
uv run python -m scripts.run_backtest --code 300308 --strategy ma_cross --start 2024-01-01
```

akshare 拉取 OHLCV（复用 `engine/agent/technical_fetcher.py` 的 fetch 逻辑），调用 `run_backtest`，打印结果。
