# Goal

定义 Phase 9 LangGraph Agent 的核心数据结构：

- `FrozenMarketSnapshot`：在会话开始时一次性捕获，所有 LangGraph 节点不可变更，
  作为防幻觉的数据锚点。
- `AnalysisState`：跨节点可变的 `TypedDict`，承载 KG/RAG 检索结果和信号输出。
- `build_snapshot(db_path)`：从 DuckDB 读取持仓和宏观状态，构造不可变快照。

# In Scope

## `engine/agent/state.py`

```python
from __future__ import annotations

import uuid
from datetime import datetime
from typing import Literal, TypedDict

from pydantic import BaseModel, Field

from engine.macro_gate import MacroState, get_macro_state
from engine.schemas import HoldingRow, PortfolioSummary, StockSignal


class FrozenMarketSnapshot(BaseModel, frozen=True):
    """Immutable market snapshot captured once per analysis session."""

    session_id: str = Field(default_factory=lambda: str(uuid.uuid4()))
    captured_at: datetime = Field(default_factory=datetime.utcnow)
    holdings: tuple[HoldingRow, ...]
    macro_state: Literal["green", "yellow", "red"]
    portfolio_summary: PortfolioSummary
    price_snapshot: dict[str, float]      # code → last close price
    change_pct_snapshot: dict[str, float] # code → change_pct_1d


class AnalysisState(TypedDict):
    snapshot: FrozenMarketSnapshot
    kg_subgraph: dict[str, list[str]]   # code → KG context strings
    rag_chunks: dict[str, list[str]]    # code → RAG text excerpts
    signals: list[StockSignal]
    reasoning: dict[str, str]           # code → AI reasoning text
    errors: list[str]
    session_id: str


def build_snapshot(db_path: str) -> FrozenMarketSnapshot:
    """Read DuckDB holdings and macro state, return immutable snapshot.

    Uses engine.portfolio.load_holdings() for holding rows and
    engine.macro_gate.get_macro_state() for current macro signal.
    price_snapshot and change_pct_snapshot populated from holdings
    current_price and pnl_pct fields (no intraday fetch).
    """
    from engine.portfolio import load_holdings

    holdings = load_holdings(db_path)
    macro = get_macro_state()
    from engine.portfolio import compute_portfolio_summary

    summary = compute_portfolio_summary(holdings)
    price_snap = {h.code: h.current_price for h in holdings}
    change_snap: dict[str, float] = {}
    for h in holdings:
        try:
            change_snap[h.code] = float(h.pnl_pct.rstrip("%"))
        except (ValueError, AttributeError):
            change_snap[h.code] = 0.0

    return FrozenMarketSnapshot(
        holdings=tuple(holdings),
        macro_state=macro.value,
        portfolio_summary=summary,
        price_snapshot=price_snap,
        change_pct_snapshot=change_snap,
    )
```

# Implementation Notes

- `frozen=True` 通过 Pydantic v2 的 `model_config` 实现，任何字段赋值都抛 `ValidationError`
- `holdings` 使用 `tuple` 而非 `list`，使得 Pydantic frozen model 的不变性语义更明确
- `build_snapshot` 中 `price_snapshot` 直接用 `current_price`（DuckDB 当日收盘价）而非
  实时行情，避免引入 intraday fetch 复杂度
- `pnl_pct` 字段存储为字符串（如 `"+2.3%"`），需手动解析为 float
