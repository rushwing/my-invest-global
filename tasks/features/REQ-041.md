## Summary

在仪表盘顶部添加全局操作栏（GlobalActionBar），提供刷新行情和运行 AI 分析的页面触发按钮，
替代原来只能通过 CLI 操作的入口。同时在 Tab 2 底部新增 T+1 调仓摘要条。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/global-action-bar.jsx`：
- 市场状态指示灯（LIVE / 盘后 / 数据异常）
- `↻ 刷新行情` ghost 按钮 → 后台线程执行 `refresh-data-agent --once`
- `▶ 运行分析` primary 按钮 → 后台线程执行 `scripts.run_analysis --once`
- 时间戳（最近刷新/分析时间）
- `🔒 锁定` 按钮

`docs/design/dashboard/design_handoff_dashboard/tab2-brief-v2.jsx`：
- T+1 调仓摘要条（今日可操作 / 建议减仓 N 只 / 建议加仓 N 只 / T+1 锁仓 N 只）
- `→ 进入调仓分析` CTA

## Acceptance Criteria

- `trigger_refresh()` 和 `trigger_analysis()` 在后台线程执行，不阻塞主线程
- 任务运行中：对应按钮 disabled + loading 文案（"⟳ 刷新中…" / "⟳ 分析中…"）
- 运行中时每 2 秒自动轮询 `get_status()` 并 rerun，完成后停止轮询
- 错误状态：状态点变红，时间戳区显示"刷新失败"/"分析失败"提示
- 市场状态：CST (UTC+8) 工作日 09:30–11:30 / 13:00–15:00 → LIVE（绿）；否则盘后（灰）
- `app/utils/runner.py`：使用模块级 `_STATUS` dict（非 `st.session_state`）跨线程共享状态
- Tab 2 T+1 摘要条：从 `load_latest_signals()` 统计 reduce/stop_loss、buy/add/strong_add 数量
