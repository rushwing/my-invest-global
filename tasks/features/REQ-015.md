## Summary

可收起侧边栏，支持用户直接在浏览器中录入/修改持仓数据，弥补券商 API 不开放的缺口。收起态显示56px头像轨；展开态提供完整的持仓编辑表格、CSV导入、保存/撤销功能。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/holdings-editor.jsx`：
- 收起态（56px）：头像圆圈 + 状态点 + 脏数据指示点
- 展开态（420px）：状态栏 / CSV导入区 / 可编辑表格 / 汇总栏

`docs/design/dashboard/design_handoff_dashboard/holdings-editor-auth.jsx`：认证状态变化对侧边栏的影响

## Acceptance Criteria

- 点击头像圆圈 → 侧边栏在 56px（收起）和 420px（展开）之间切换
- 修改 `cost_price` 后：`market_value` 列自动更新 = `current_price × quantity`（只读）
- 头像上显示脏数据指示点（橙色小圆）当且仅当有未保存修改
- 点击保存 → 写入 `data/agent_input/cn/holdings_{YYYYMMDD}.csv` → `st.rerun()`
- 点击撤销 → 恢复到上次保存快照，脏数据点消失
- CSV导入：上传合法 CSV → 预览行数 → Apply → 覆盖当前编辑器数据
- CSV导入：列名/格式不符 → 显示错误，不覆盖当前数据
- 未解锁（REQ-014）时：编辑器显示禁用态，Save/Apply 按钮不可点

## Implementation Details

### State Management

```python
# session_state 键
st.session_state["sidebar_open"]      # bool，控制收起/展开
st.session_state["holdings_editor"]   # pd.DataFrame，当前编辑中的数据
st.session_state["holdings_saved"]    # pd.DataFrame，上次保存的快照
st.session_state["holdings_source"]   # str: "手动编辑" | "CSV导入" | "OCR待校正"
st.session_state["holdings_date"]     # date，快照日期
```

### 可编辑列配置

```python
column_config = {
    "code":          st.column_config.TextColumn("代码"),
    "name":          st.column_config.TextColumn("名称"),
    "category":      st.column_config.SelectboxColumn("类别", options=["白马股", "弹性股"]),
    "cost_price":    st.column_config.NumberColumn("成本价", format="%.2f"),
    "current_price": st.column_config.NumberColumn("现价", format="%.2f"),
    "quantity":      st.column_config.NumberColumn("数量", step=1),
    "market_value":  st.column_config.NumberColumn("市值", disabled=True, format="%.0f"),
}
```

`market_value` 在每次 data_editor 回调后自动计算：`df["market_value"] = df["current_price"] * df["quantity"]`

### 来源徽章

### 保存逻辑

```python
def save_holdings(df: pd.DataFrame, date: date) -> Path:
    path = Path(f"data/agent_input/cn/holdings_{date:%Y%m%d}.csv")
    path.parent.mkdir(parents=True, exist_ok=True)
    df.to_csv(path, index=False)
    st.session_state["holdings_saved"] = df.copy()
    st.session_state["holdings_date"] = date
    return path
```

### 汇总栏告警

```python
# 单只弹性股超过 8% 总市值时显示红色告警
elastic_positions = df[df["category"] == "弹性股"]
total_mv = df["market_value"].sum()
for _, row in elastic_positions.iterrows():
    pct = row["market_value"] / total_mv * 100
    if pct > 8:
        st.markdown(f'⚠ {row["name"]} {pct:.1f}% > 8% 上限', unsafe_allow_html=True)
```
