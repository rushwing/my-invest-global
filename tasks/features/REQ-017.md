## Summary

实现 Tab 2 "每日策略简报"：宏观横幅、可折叠策略简报、综合信号评分表、美股先行图和美→中映射面板。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/tab2-brief.jsx`：
- `MacroBanner`：左边框 pill + 宏观描述 + US chip 徽章
- `StrategyBrief`：3 个 st.expander（市场判断/个股操作/风险提示）
- `ScoreTable`：A/B/C ProgressColumn + 综合分数着色 + stop_loss 行高亮
- `USLeadingChart`：水平双向柱状图（center=0）
- `MappingPanel`：美→中板块表

## Acceptance Criteria

- `recommendations_{date}.md` 内容渲染在 StrategyBrief "市场判断" expander 中
- ScoreTable A/B/C 列使用 `st.column_config.ProgressColumn`（min=0, max=100）
- stop_loss 行显示红色背景（`#E84040` 15% 透明度）
- scoreColor 阈值：≥75=bull (#00C47A)；≥60=accent (#4F8EF7)；≥40=textMuted (#9AA0AC)；<40=bear (#E84040)
- LLM 归因显示 `claude-sonnet-4-6`（不是 GPT-4）
- MacroBanner 宏观状态颜色：green→#00C47A；yellow→#F5A623；red→#E84040

### ScoreTable

```python
st.dataframe(
    df,
    column_config={
        "signal_a": st.column_config.ProgressColumn("技术 A", min_value=0, max_value=100),
        "signal_b": st.column_config.ProgressColumn("基本面 B", min_value=0, max_value=100),
        "signal_c": st.column_config.ProgressColumn("情绪 C", min_value=0, max_value=100),
        "composite": st.column_config.NumberColumn("综合", format="%d"),
        "action":    st.column_config.TextColumn("建议"),
    },
    use_container_width=True,
)
```

stop_loss 行着色：通过 `df.style.apply()` 对 action="stop_loss" 的行设置红色背景。

scoreColor 函数（Python 实现）：
```python
def score_color(score: float) -> str:
    if score >= 75: return "#00C47A"
    if score >= 60: return "#4F8EF7"
    if score >= 40: return "#9AA0AC"
    return "#E84040"
```
