## Summary

实现 Tab 1 "持仓总览"：页头状态行、AllocationDonut 环形图、KPI 三卡片、HoldingsHeatmap Treemap 热力图、DeviationTable 偏差表。数据源为最新 HoldingsFile。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/tab1-holdings.jsx`：
- `AllocationDonut`：SVG 环形图 → 实现用 `px.pie(hole=0.65)`
- `HoldingsHeatmap`：squarify Treemap → 实现用 `px.treemap()`
- `DeviationTable`：条件着色偏差表
- `KPIRow`：3 列 KPI 卡片 + 进度条 + 目标标记线

## Acceptance Criteria

- `check_portfolio_balance()` 计算的偏差值与 DeviationTable 的偏差列完全一致
- AllocationDonut 内圈文字显示当前 白马/弹性 实际占比
- HoldingsHeatmap 颜色 scale：bear (#E84040) at -20% → neutral (#F5A623) at 0 → bull (#00C47A) at +20%
- DeviationTable 行着色：|dev| > 5% → 黄色背景；|dev| > 10% → 红色背景；按 |偏差| 降序排列
- 最大单一弹性股 KPI 卡片：当前占比 > 8% → 显示红色
- Tab 1 在 HoldingsFile 不存在时显示友好提示，不崩溃

## Components

### 页头

```
[2026-05-17]  [● A股 交易中]  总市值: ¥1,234,567  总盈亏: +¥87,654 (+7.6%)
```

- 市场状态：工作日 09:30–15:00 → "● A股 交易中"（绿）；否则 "○ A股 休市"（灰）
- 盈亏颜色：正值=bull，负值=bear

### AllocationDonut

- `px.pie(values=[白马MV, 弹性MV], hole=0.65)`
- 内圈覆盖文字：当前 白马% / 弹性%
- 偏差徽章（右上角）：|dev| < 5% → bull "目标: 67/33"；5–10% → neutral "⚠ 偏离"；> 10% → bear "⚠ 需调仓"

### KPI 三卡片

| 卡片 | 数值 | 说明 |
|---|---|---|
| 白马市值 | ¥X,XXX,XXX | 含占比 % |
| 弹性市值 | ¥X,XXX,XXX | 含占比 % |
| 最大单一弹性 | XX.X% | > 8% 时 bear 颜色，显示股票名 |

每卡片底部有进度条（当前→目标），目标刻度线用竖线标出。

### HoldingsHeatmap

```python
fig = px.treemap(
    df,
    path=["category", "name"],
    values="market_value",
    color="pnl_pct",
    color_continuous_scale=[(0, "#E84040"), (0.5, "#F5A623"), (1, "#00C47A")],
    color_continuous_midpoint=0,
    custom_data=["code", "pnl_pct"],
)
fig.update_traces(
    texttemplate="%{label}<br>%{customdata[0]}<br>%{customdata[1]:.1f}%",
    textfont_family="JetBrains Mono",
)
```

### DeviationTable

列：代码 | 名称 | 类别 | 市值 | 当前占比 | 目标占比 | 偏差 | 建议

```python
def style_deviation_row(val):
    abs_dev = abs(val)
    if abs_dev > 10: return "background-color: rgba(232,64,64,0.15)"
    if abs_dev > 5:  return "background-color: rgba(245,166,35,0.15)"
    return ""

df_styled = df.sort_values("abs_dev", ascending=False).style.applymap(
    style_deviation_row, subset=["偏差"]
)
st.dataframe(df_styled, use_container_width=True)
```
