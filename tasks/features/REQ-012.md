# Goal

实现 `engine/mcp/macro_tools.py`，向 ADR-006 Phase 7 MCP Server 注册 6 个
宏观工具，供 Claude Desktop 交互和 Phase 9 LangGraph ReAct Agent 调用。
工具仅读取 DuckDB，业务逻辑留在 `regime.py`。

# In Scope

## `engine/mcp/macro_tools.py`

所有工具使用 Pydantic 模型定义返回值结构（ADR-006 防幻觉机制二）。

### 工具 1：`get_macro_state`

```python
class MacroStateResult(BaseModel):
    composite_state: Literal["green", "yellow", "red", "stale"]
    capex_state: Literal["green", "yellow", "red", "stale"]
    yield_curve_state: Literal["normal", "flat", "inverted", "stale"]
    risk_state: Literal["risk_on", "risk_off", "neutral", "stale"]
    capex_as_of: str           # "2026-Q1"
    yield_as_of: date | None
    computed_at: datetime | None
    is_stale: bool             # True if composite_state == "stale"

@mcp.tool()
def get_macro_state() -> dict:
    """返回当前宏观门控状态（CapEx / 收益率曲线 / 风险偏好 / 综合）。"""
```

### 工具 2：`get_capex_trend`

```python
class CapExQuarter(BaseModel):
    company: str
    fiscal_quarter: str
    period_end: date
    capex_usd: float | None    # 单季，billion USD
    capex_yoy_pct: float | None
    filing_form: str | None

class CapExTrendResult(BaseModel):
    company: str
    quarters: list[CapExQuarter]
    latest_qoq_pct: float | None
    is_stale: bool

@mcp.tool()
def get_capex_trend(company: str, quarters: int = 4) -> dict:
    """返回 MSFT/AMZN/GOOGL/META 最近 N 季 CapEx 及 QoQ 增速（附时效标注）。"""
```

### 工具 3：`get_fomc_calendar`

class FOMCMeeting(BaseModel):

class IndexSnapshot(BaseModel):

# Acceptance Criteria

- `TC-012-01 get_macro_state stale 传播`：mock `get_regime_latest()` 返回
  `composite_state="stale"`，工具响应 `is_stale=True`
- `TC-012-02 get_capex_trend Pydantic 校验`：mock storage 返回完整 4 季数据，
  工具返回值经 `CapExTrendResult(**result)` 不抛异常
- `TC-012-03 get_fomc_calendar 过滤`：预填充 3 条 FOMC 记录（1 条 60 天内，
  2 条 120 天后），`get_fomc_calendar(lookahead_days=90)` 仅返回 1 条
- `TC-012-04 get_yield_curve realtime_start`：mock `get_latest_indicator("DGS10")`
  返回含 `realtime_start="2026-04-01"` 的记录，工具响应含相同 vintage 字段
- `TC-012-05 get_news_sentiment 预算显示`：mock `get_av_budget()` 返回 18，
  工具响应 `av_budget_remaining=7`（25-18）
- `TC-012-06 get_global_indices group 过滤`：`get_global_indices(["J"])` 仅
  返回 Group J（大宗商品）的指标，不混入 Group K 数据

# Implementation Notes

- 所有工具返回 `dict`（MCP 协议要求），内部先构建 Pydantic 模型再调用
  `.model_dump(mode="json")` 序列化
- `get_news_sentiment()` 中的 `av_budget_remaining = max(0, 25 - get_av_budget(today))`
- `hours_back` 过滤：`macro_indicators` 中 `group_code="O"` 且
  `utc_ts >= now - timedelta(hours=hours_back)` 的记录
- Phase 9 集成备注：LangGraph 中使用 `get_macro_state()["is_stale"]` 决定是否
  将宏观信号标记为 `[stale]` 并降低其对 portfolio 决策的权重
