## Summary

实现 Tab 4 "信号仪表盘"：宏观周期面板（三色交通灯 + Big-4 云 CapEx 表）、板块紧缺度矩阵、仓位再平衡仪表盘（含触发器状态表）和历史再平衡时间线。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/tab4-signals.jsx`：
- `MacroCyclePanel`：3 态交通灯（发光效果）+ CapEx 表
- `ScarcityMatrix`：Tier 1–4 四列 chip 网格
- `Gauge`：半圆形 SVG 仪表盘（设计用 SVG）→ 实现用 `streamlit.components.v1.html`
- `RebalanceTriggers`：7 行触发器状态表
- `HistoryTimeline`：st.dataframe + 可选 Plotly scatter

## Acceptance Criteria

- MacroStateFile state=GREEN → 弹性仓位仪表盘目标显示 38%；state=RED → 显示 20%
- Gauge ±5% 区域颜色变化（容忍带）；±10% 触发器触发（两个不同阈值，不混淆）
- ScarcityMatrix chip 颜色：composite_score ≥ 70 → bull (#00C47A)；50–70 → neutral (#F5A623)；< 50 → bear (#E84040)
- ◉ 符号 = 当前持有股票；⚠ 符号 = warn=True 的股票
- RebalanceTriggers 表：status="red" 行显示红色背景
- 手动宏观覆盖按钮调用 `set_macro_state_override()`（REQ-001）并触发 st.rerun()

### ScarcityMatrix

4 列 `st.columns(4)`，每列一个 Tier：

```
Tier-1 (橙色)    Tier-2 (黄色)    Tier-3 (灰色)    Tier-4 (蓝色)
┌───────────┐   ┌───────────┐   ...
│ ◉中际旭创  │   │  华为海思  │
│ 300308    │   │  (未上市)  │
│ score: 84 │   │ score: 78 │
└───────────┘
```

chip 颜色由 `composite_score` 决定：
```python
def chip_color(score: int) -> str:
    if score >= 70: return "#00C47A"
    if score >= 50: return "#F5A623"
    return "#E84040"
```

◉ = `owned=True`（当前持有）；⚠ = `warn=True`

### RebalanceTriggers + Gauge

仪表盘（半圆 SVG via `streamlit.components.v1.html`）：
- 白马：当前比例 vs 目标（宏观状态决定目标，Green→62%/Yellow→67%/Red→80%）
- 弹性：当前比例 vs 目标（Green→38%/Yellow→33%/Red→20%）
- ±5%：仪表盘颜色警告带（neutral 颜色）
- ±10%：触发器触发条件（rebalance fires）

```python
ELASTIC_TARGETS = {"GREEN": 38, "YELLOW": 33, "RED": 20}
target = ELASTIC_TARGETS[macro_state]
current = elastic_pct * 100
deviation = current - target

# Gauge 颜色
if abs(deviation) > 10: gauge_color = "#E84040"  # 触发
elif abs(deviation) > 5: gauge_color = "#F5A623"  # 警告
else:                     gauge_color = "#00C47A"  # 正常
```

7 触发器状态表（来自 project-scope.md § Dynamic Rebalance Triggers）：

| 触发器 | 方向 | 幅度 | 状态 |
|---|---|---|---|
| 云 CapEx 指引上调（季度） | 弹性股 ↑ | +7%，上限 40% | 正常/观察/触发 |
| 云 CapEx 指引下调 >10% | 弹性股 ↓ | -13%，下限 20% | ... |
| 板块整体回撤 >15% | 白马股 ↑ | +7%，上限 75% | ... |
| 板块突破前高（放量） | 弹性股 ↑ | +5%，上限 38% | ... |
| 单只弹性股亏损 >20% | 止损提示 | 建议减仓/退出 | ... |
| 单只持仓达目标价 80% | 止盈提示 | 建议减仓 30–50% | ... |
| 单只弹性股超 10% | 调仓提示 | 建议减至 <8% | ... |

status 颜色：正常=textMuted；观察=neutral；触发=bear + 红色背景行
