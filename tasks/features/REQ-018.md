## Acceptance Criteria

- 选择 ticker 300308 → RSI(14) 曲线所有非空值在 [0, 100] 内（前 13 行 warm-up NaN 不渲染，不视为越界）
- BB upper > BB mid > BB lower（所有非空 index 点，即第 20 天起）
- 金叉标记：MACD 线由下向上穿越信号线 → 绿色圆圈
- 死叉标记：MACD 线由上向下穿越信号线 → 红色圆圈
- FundamentalCard Tier 颜色：Tier-1=#FF7A1A, Tier-2=#F5A623, Tier-3=#9AA0AC, Tier-4=#4F8EF7
- NewsTimeline 最多展示 8 条，按时间倒序
- 切换股票 → 图表数据完整重载，无残留前一只股票数据
- 日期范围切换（1M/3M/6M/1Y）→ 图表 x 轴截断正确

## compute_indicators 契约（来自 REQ-020）

`compute_indicators(df: pd.DataFrame) -> pd.DataFrame`
- 输入：含 `open/high/low/close/volume` 列、按 `date` 升序排列的 DataFrame（来自 `load_ohlcv()`）
- 返回：附加列 `ma65, bb_upper, bb_mid, bb_lower, macd, macd_signal, macd_hist, rsi14`
- 行数不足时各指标列填 NaN，不抛异常
- 不缓存（每次调用 pandas-ta 计算）

Tab 3 调用序列：
```python
ohlcv_df = load_ohlcv(ticker, days=180)          # 从 aidc.duckdb 读 / AkShare fallback
df = compute_indicators(ohlcv_df)                  # 添加技术指标列
news_df = load_news_for_ticker(ticker, limit=8)    # 从 aidc.duckdb 读
```

## FundamentalCard 数据来源

| 字段 | 数据源 | 缺失时行为 |
|---|---|---|
| `rank`（Tier 1–4） | `load_scarcity_matrix()["rank"]` 按 code 匹配 | 显示 "Tier-?" |
| `current_price` | `load_ohlcv()` 最新 close | 显示 "—" |
| `target_price` | `load_scarcity_matrix()["target_price"]`（字段不存在时为空） | 不显示上涨空间行 |
| `market_cap` | `load_scarcity_matrix()["market_cap"]`（可选） | 不显示 |
| `pe` | `load_scarcity_matrix()["pe"]`（可选） | 不显示 |
| `beta` | `load_scarcity_matrix()["beta"]`（可选） | 不显示 |
| `avg_volume` | `load_ohlcv()` 近 20 日 volume 均值 | 不显示 |
| 机构评级 | `load_scarcity_matrix()["rating"]`（字符串） | 不显示评级行 |

`load_scarcity_matrix()` 返回 DataFrame 中 code 不匹配时，FundamentalCard 仅显示 ticker 和现价，其余字段隐藏（不抛异常）。
