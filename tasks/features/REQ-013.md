## Summary

通过 `st.markdown(unsafe_allow_html=True)` 将设计 token、Google Fonts 和全局覆盖样式注入 Streamlit，使所有组件的颜色、字体、间距与高保真设计稿精确匹配。

## Design Reference

`docs/design/dashboard/design_handoff_dashboard/design-system.jsx` — TOKENS 对象 + SIGNAL enum + 可复用组件定义。

## Acceptance Criteria

- `streamlit run app/dashboard.py` 渲染页面背景为 `#0E1117`（非 Streamlit 默认白/深蓝）
- 所有数字元素（价格、百分比、分数）显示 JetBrains Mono 字体，`font-feature-settings: "tnum"`
- Noto Sans SC 作为 UI 字体（中文 + 拉丁文）加载
- 卡片 padding 为 12px（覆盖 Streamlit 默认 16px）
- 颜色 token 可通过 CSS 变量在组件层引用（`var(--bull)`, `var(--bear)` 等）

```
bg:         #0E1117

```

## Deliverables

- `app/styles.py`：`inject_global_css()` 函数，在 `dashboard.py` `main()` 最顶部调用
- 内容包含：
  - Google Fonts `<link>` 标签（Noto Sans SC 400/500/600/700, JetBrains Mono 400/500/600）
  - CSS `:root` 变量（所有 token）
  - 全局覆盖（body 背景、字体、Streamlit 容器 padding）
  - `.mono-num` utility 类（tabular-nums + JetBrains Mono）
  - `.kpi-card`, `.signal-badge`, `.tag` 基础样式类
- `app/components/ui.py`：Python helper 函数
  - `kpi_card(label, value, delta=None, delta_color="bull")` → `st.markdown(html)`
  - `signal_badge(signal_key)` → inline HTML span
  - `tag(text, color)` → inline HTML span
  - `mono_num(value, prefix="", suffix="")` → inline HTML span with `.mono-num`

## Implementation Notes

- 使用 `unsafe_allow_html=True`；不引入外部 CSS 文件（Streamlit 不支持静态文件直接引用）
- `inject_global_css()` 必须在 `st.set_page_config()` 之后、任何 `st.` 渲染调用之前执行
- tc_policy = exempt（纯视觉，无业务逻辑，不需要自动化测试）
