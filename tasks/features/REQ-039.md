# Goal

在仪表盘中为持仓标的提供筹码分布可视化面板，包含水平直方图和关键指标卡片。
由于 Tab 3（REQ-018）尚未实现，本 REQ 先作为独立 Streamlit 页发布（`app/pages/chip_panel.py`），
待 Tab 3 实现后（REQ-018）无缝集成。

# In Scope

## `app/pages/chip_panel.py`

### 页面入口

```python
def chip_panel_page() -> None:
    """独立 Streamlit 页，可通过 st.navigation 或直接 streamlit run 访问。"""
    st.title("筹码分析")
    code = st.selectbox("选择标的", load_holdings_codes())
    current_price = st.number_input("当前价格", min_value=0.01, value=0.0, format="%.2f")
    if st.button("分析"):
        _render_chip_panel(code, current_price)
```

### 核心渲染函数

```python
def render_chip_panel(
    code: str,
    current_price: float,
    analysis: ChipAnalysis,             # 信号字段来源
    summary: ChipSummary | None = None, # 直方图原始数据；None 时跳过图表渲染并显示提示
) -> None:
    """纯渲染函数，便于 Tab 3 集成时直接调用。
    summary.bars 用于绘制直方图；analysis 用于 ChipSummaryCard 和信号状态卡片。"""
```

> `summary` 为 `None` 时，直方图区域显示 `st.info("暂无筹码直方图数据")`，
> ChipSummaryCard 和信号状态卡片仍正常渲染（仅依赖 analysis）。

### 组件 1：水平筹码直方图

数据来源：`summary.bars`（`list[ChipBar]`，来自 REQ-036）。
使用 `plotly.graph_objects.Bar` + `orientation='h'`：

- X 轴：`chip_ratio`（百分比，右→）
- Y 轴：价格档位中点（`(price_lower + price_upper) / 2`）
- 颜色规则：
  - 档位中点 < `current_price`：`#00C47A`（获利区，绿色）
  - 档位中点 ≥ `current_price`：`#E84040`（套牢区，红色）
- 标注线（`plotly shapes`）：
  - 实线橙色：`current_price`，标签 "现价"
  - 虚线白色：`avg_cost`，标签 "均成本"
  - 虚线灰色：`range_90_lower` / `range_90_upper`，标签 "90%区间"
- 布局：深色背景（`bg=#0E1117`），高度 400px，Y 轴标签右对齐

### 组件 2：ChipSummaryCard

4 列 `st.metric` 展示：

| 列 | 指标 | 样式规则 |
|---|---|---|
| 收盘获利 | `f"{analysis.profitable_pct:.1%}"` | ≥ 95% 显示 ⚠ delta（获利盘压力大） |
| 平均成本 | `f"¥{analysis.avg_cost:.2f}"` | 偏离率 `cost_deviation_pct` 作 delta |
| 筹码集中度 | `f"{analysis.concentration:.1f}%"` | `chip_lock_level` 枚举对应中文标签 |
| 信号状态 | 见下方 | |

**信号状态卡片：**

```python
if analysis.above_90_band:
    st.error(f"⚠ 突破90%筹码上界（{analysis.range_90_upper:.2f}）— 获利盘兑现压力")
elif analysis.below_90_band:
    st.warning(f"价格低于90%筹码下界（{analysis.range_90_lower:.2f}）— 套牢密集区")
else:
    st.info(f"价格在90%筹码区间内（{analysis.range_90_lower:.2f}–{analysis.range_90_upper:.2f}）")
```
