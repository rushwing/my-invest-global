# Goal

在 LangGraph pipeline 中插入 `technical_fetcher` 节点，通过 akshare 拉取 A 股 OHLCV 并用 pandas-ta 计算 MA5/10/20/30、MACD(DIF/DEA/柱)、RSI(14)。技术指标以 T-编号数据源（T1..T8）注入 `signal_ranker` prompt，直接提升 `technical_score` 的可信度。

# In Scope

## `engine/agent/technical_fetcher.py`（新建）

```python
import akshare as ak
import pandas as pd
import pandas_ta as ta
from datetime import date, timedelta

def fetch_technicals(
    codes: list[str],
    lookback_days: int = 60,
) -> dict[str, dict[str, float]]:
    """Fetch OHLCV from akshare, compute MA/MACD/RSI. Returns {} per code on failure."""
    result: dict[str, dict[str, float]] = {}
    start = (date.today() - timedelta(days=lookback_days + 30)).strftime("%Y%m%d")
    for code in codes:
        try:
            df = ak.stock_zh_a_hist(
                symbol=code, period="daily", start_date=start, adjust="qfq"
            )
            df = df.rename(columns={"收盘": "Close", "开盘": "Open", "最高": "High",
                                     "最低": "Low", "成交量": "Volume"})
            df.ta.sma(length=5, append=True)
            df.ta.sma(length=10, append=True)
            df.ta.sma(length=20, append=True)
            df.ta.sma(length=30, append=True)
            df.ta.macd(fast=12, slow=26, signal=9, append=True)
            df.ta.rsi(length=14, append=True)
            last = df.iloc[-1]
            result[code] = {
                "MA5": round(float(last.get("SMA_5", float("nan"))), 2),
                "MA10": round(float(last.get("SMA_10", float("nan"))), 2),
                "MA20": round(float(last.get("SMA_20", float("nan"))), 2),
                "MA30": round(float(last.get("SMA_30", float("nan"))), 2),
                "MACD_DIF": round(float(last.get("MACD_12_26_9", float("nan"))), 3),
                "MACD_DEA": round(float(last.get("MACDs_12_26_9", float("nan"))), 3),
                "MACD_BAR": round(float(last.get("MACDh_12_26_9", float("nan"))), 3),
                "RSI14": round(float(last.get("RSI_14", float("nan"))), 1),
            }
        except Exception:
            result[code] = {}
    return result
```

## `engine/agent/nodes.py` — technical_fetcher 节点

```python
def technical_fetcher(state: AnalysisState) -> dict[str, Any]:
    from engine.agent.technical_fetcher import fetch_technicals
    codes = [h.code for h in state["snapshot"].holdings]
    try:
        data = fetch_technicals(codes)
    except Exception as e:
        return {"technical_data": {}, "errors": state["errors"] + [f"technical_fetcher: {e}"]}
    return {"technical_data": data}
```

`signal_ranker` 读取 `technical_data`，在构建每只股票的 source_index 时追加 T-refs：

```
T1  [技术] MA5 = {v}
T2  [技术] MA10 = {v}
T3  [技术] MA20 = {v}
T4  [技术] MA30 = {v}
T5  [技术] MACD DIF = {v}   ← 用户术语: MACDIF
T6  [技术] MACD DEA = {v}   ← 用户术语: MACDFS
T7  [技术] MACD柱 = {v}     ← 用户术语: MACDIE (bar = (DIF-DEA)×2)
T8  [技术] RSI(14) = {v}
```

并在 system prompt 补充均线/MACD 解读规则：
```
均线系统：MA5>MA10>MA20 看多排列，MA5<MA20 短期偏弱
MACD：DIF>DEA 且柱>0 为金叉（多头），DIF<DEA 且柱<0 为死叉（空头）
```

- NaN 值（数据不足时）保留为 float("nan")，T-ref 显示时跳过 NaN 条目
