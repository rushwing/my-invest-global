## Objective

验证 `FrozenMarketSnapshot` 作为 Pydantic `BaseModel` 可以通过 `model_dump()` 序列化为 dict，且 `holdings` 字段序列化为 list。

## Preconditions

- `engine.agent.state.FrozenMarketSnapshot` 可正常 import
- 构造一个含 1 个 `HoldingRow` 的 legally valid snapshot

## Steps

1. 构造 `FrozenMarketSnapshot` 实例（`holdings` 为 `tuple` 含 1 个 `HoldingRow`）
2. 调用 `d = snap.model_dump()`
3. 调用 `j = snap.model_dump_json()`

## Expected Result

- `d` 类型为 `dict`，不抛异常
- `d["session_id"]` 为字符串，等于构造时的 session_id
- `d["holdings"]` 类型为 `list`（Pydantic 默认序列化 tuple→list）
- `len(d["holdings"])` == 1
- `d["macro_state"]` 为字符串（如 `"green"`）
- `j` 类型为 `str`，是合法 JSON（`json.loads(j)` 不抛异常）
- `d["captured_at"]` 为 ISO 8601 字符串

## Notes

- `model_dump(mode="json")` 可进一步控制 `datetime` → ISO 字符串的行为
- 此 TC 是 LangGraph checkpoint 序列化（若需手动 checkpoint）的前提验证
