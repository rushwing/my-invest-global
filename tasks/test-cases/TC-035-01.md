## Objective

验证 `run_backtest(code="300308", strategy="ma_cross", df=mock_ohlcv_df)` 返回符合 REQ-035 口径的 `BacktestResult`，并覆盖基础输入校验与无交易 fallback。

## Preconditions

- `engine.backtest.runner.run_backtest` 和 `BacktestResult` 可 import
- `mock_ohlcv_df` 为至少 40 行的 `pd.DataFrame`
- `mock_ohlcv_df` index 为 `pd.DatetimeIndex`，列包含小写 `open/high/low/close/volume`
- `close` 序列需构造出至少一次 MA5 上穿/下穿 MA20 的机会

## Steps

1. 构造 40 行以上 mock OHLCV：
   ```python
   import numpy as np
   import pandas as pd

   dates = pd.date_range("2024-01-01", periods=45, freq="D")
   close = np.r_[np.linspace(100, 92, 20), np.linspace(93, 125, 25)]
   df = pd.DataFrame({
       "open": close - 0.5,
       "high": close + 1.0,
       "low": close - 1.0,
       "close": close,
       "volume": [1_000_000] * len(close),
   }, index=dates)
   ```
2. 调用：
   ```python
   result = run_backtest(code="300308", strategy="ma_cross", df=df)
   ```
3. 验证返回结构和核心字段：
   ```python
   assert isinstance(result, BacktestResult)
   assert result.code == "300308"
   assert result.strategy == "ma_cross"
   assert isinstance(result.total_return, float)
   assert isinstance(result.benchmark_return, float)
   assert isinstance(result.max_drawdown, float)
   assert isinstance(result.win_rate, float)
   assert isinstance(result.num_trades, int)
   assert isinstance(result.equity_curve, pd.Series)
   assert len(result.equity_curve) == len(df)
   assert result.equity_curve.index.equals(df.index)
   assert result.equity_curve.iloc[0] == 1.0
   ```
4. 验证输入边界：
   ```python
   with pytest.raises(ValueError, match="Empty OHLCV data"):
       run_backtest("300308", "ma_cross", df.iloc[0:0])

   with pytest.raises(KeyError, match="close column required"):
       run_backtest("300308", "ma_cross", df.drop(columns=["close"]))

   with pytest.raises(ValueError, match="NaN values in close column"):
       run_backtest("300308", "ma_cross", df.assign(close=[float("nan")] + list(df["close"].iloc[1:])))

   with pytest.raises(ValueError, match="Insufficient data"):
       run_backtest("300308", "ma_cross", df.iloc[:25])

   with pytest.raises(ValueError, match="Unknown strategy"):
       run_backtest("300308", "bad_strategy", df)
   ```
5. 构造不会触发交易的平坦价格序列，验证无交易 fallback：
   ```python
   flat = df.assign(close=100.0)
   result = run_backtest("300308", "ma_cross", flat)
   assert result.total_return == 0.0
   assert result.max_drawdown == 0.0
   assert result.win_rate == 0.0
   assert result.num_trades == 0
   assert (result.equity_curve == 1.0).all()
   ```

## Expected Result

- `ma_cross` 返回 `BacktestResult`
- `equity_curve` 是与输入同长度、同 index 的 `pd.Series`，起始净值为 `1.0`
- 收益、回撤、胜率字段均为 float；`num_trades` 为 int
- 空 df、缺少 `close`、`close` 含 NaN、数据不足、未知策略均抛出指定异常
- 无交易信号时返回零收益、零回撤、零胜率、零交易次数、净值恒为 1.0

## Notes

- 此 TC 不依赖 akshare 或 Streamlit
- mock 数据必须使用小写列名，匹配 REQ-035 的 OHLCV 契约
- 实现应使用 `signal.shift(1)` 避免 lookahead bias；该细节可通过后续代码测试检查交易日收益是否从下一日开始生效
