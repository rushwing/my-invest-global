## Objective

验证 `ChipLockLevel` 阈值：`<40` 为 LOW，`40-60` 为 MEDIUM，`>=60` 为 HIGH。

## Preconditions

- `analyze_chip` 和 `ChipLockLevel` 可 import
- 可构造只改变 `concentration` 的 mock `ChipSummary`

## Steps

1. 分别构造 `concentration=33.92`、`55.0`、`65.0` 的 summary。
2. 调用 `analyze_chip`。
3. 断言：
   ```python
   assert low.chip_lock_level == ChipLockLevel.LOW
   assert medium.chip_lock_level == ChipLockLevel.MEDIUM
   assert high.chip_lock_level == ChipLockLevel.HIGH
   ```

## Expected Result

- 33.92 → LOW
- 55.0 → MEDIUM
- 65.0 → HIGH

## Notes

- 边界值 40.0 和 60.0 可在实现测试中追加
