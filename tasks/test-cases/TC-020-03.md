## Objective

验证 REQ-020 中所有缓存型数据加载函数均通过 `@st.cache_data(ttl=300)` 包装。

## Preconditions

- `app.data_loader` 可 import
- 缓存函数列表包括：`load_latest_holdings`, `load_latest_signals`, `load_recommendations`, `load_ohlcv`, `load_news_for_ticker`, `load_macro_state`, `load_fund_flow_5d`, `load_market_breadth`, `load_cloud_capex`, `load_scarcity_matrix`, `load_rebalance_history`

## Steps

1. monkeypatch 底层 IO 函数或文件读取计数器。
2. 对同一参数连续调用同一 loader 两次。
3. 断言第二次调用未重复触发底层 IO。
4. 如实现可 introspect cache wrapper，则额外断言 ttl 为 300。

## Expected Result

- 每个缓存型 loader 在相同参数 300 秒内复用缓存结果
- 纯计算函数 `compute_portfolio_metrics`, `compute_indicators`, `compute_trigger_status` 不要求缓存

## Notes

- 可逐个 loader 参数化实现此 TC
