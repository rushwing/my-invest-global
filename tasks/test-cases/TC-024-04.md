## Objective

验证 `_is_write_cypher()` 能区分只读 `MATCH ... RETURN` 与含写关键字的 Cypher。

## Preconditions

- 可 import `engine.kg.query._is_write_cypher`

## Steps

1. 调用 `_is_write_cypher("MATCH (n) RETURN n")`。
2. 调用 `_is_write_cypher("MATCH (n) DELETE n")`。
3. 调用 `_is_write_cypher("MATCH (n) SET n.x=1")`。

## Expected Result

- 只读 `MATCH` 返回 `False`
- 含 `DELETE` 返回 `True`
- 含 `SET` 返回 `True`
