## Objective

验证 `verify_signals([], snapshot)` 传入空信号列表时返回空 `list`（不抛异常，不 panic）。

## Preconditions

- `engine.agent.claim_verifier.verify_signals` 可 import

## Steps

1. 构造任意合法 `FrozenMarketSnapshot`
2. 调用 `violations = verify_signals([], snapshot)`

## Expected Result

- `violations` == `[]`
- 函数正常返回，无异常抛出

## Notes

- 空 signals 场景发生在 `signal_ranker` LLM 返回空 JSON 数组或解析失败时
- 此 TC 防止在边界情况下 NPE 或 IndexError
