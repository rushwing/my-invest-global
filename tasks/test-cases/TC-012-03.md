## Objective

验证 FOMC MCP 工具使用 storage 的窗口过滤语义。

## Preconditions

- 使用 `MacroStorage(":memory:")`
- 预填充 3 条 FOMC 记录：1 条 60 天内，2 条 120 天后

## Steps

1. 调用 `get_fomc_calendar(lookahead_days=90)`。
2. 用 `FOMCCalendarResult(**result)` 校验。

## Expected Result

- `meetings` 只包含 60 天内那一条
- `next_meeting_date` 等于该会议日期
- 不返回 120 天后的会议
