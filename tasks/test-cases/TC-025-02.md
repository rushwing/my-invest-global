## Objective

验证 `RAGSchema.create_table(lancedb.connect(":memory:"))` 不抛异常，并返回可写入的
LanceDB table 对象。

## Preconditions

- 安装 `lancedb` 可选依赖
- 使用 `lancedb.connect(":memory:")`，不读写磁盘

## Steps

1. 构造 memory db。
2. 调用 `RAGSchema.create_table(db)`。

## Expected Result

- 调用不抛异常
- 返回对象有 `add` 方法
- 表名为 `RAGSchema.TABLE_NAME`
