## Objective

验证 Tab 3 图表构建函数在 `macd` 由下向上穿越 `macd_signal` 的日期添加绿色圆形 marker。

## Preconditions

- `app.components.tab3_stock.build_stock_depth_figure` 或等价图表构建函数可 import
- 准备 DataFrame，其中某日满足 `macd > macd_signal` 且前一日 `macd <= macd_signal`

## Steps

1. 构造含 `date/open/high/low/close/volume/macd/macd_signal/macd_hist/rsi14/bb_*` 的 DataFrame。
2. 调用 `fig = build_stock_depth_figure(df)`。
3. 在 `fig.data` 中找到 `name == "金叉"` 的 trace。
4. 断言：
   ```python
   assert golden_trace.marker.color == "#00C47A"
   assert golden_trace.marker.symbol == "circle"
   assert expected_cross_date in list(golden_trace.x)
   ```

## Expected Result

- 金叉 trace 存在
- marker 为绿色圆形
- marker 日期等于 MACD 上穿信号线的日期

## Notes

- 不启动 Streamlit server，仅检查 Plotly Figure
