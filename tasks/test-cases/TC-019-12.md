## Objective

验证 MacroCyclePanel 顶部左图将 20 日持仓主力净流入按 `trade_date` 汇总，并生成 `go.Bar`。

## Preconditions

- `app.components.tab4_signals.build_fund_flow_chart` 或等价函数可 import
- 准备多个 code、20 个交易日的 `fund_flow_df`

## Steps

1. 调用 `fig = build_fund_flow_chart(fund_flow_df)`。
2. 断言：
   ```python
   assert len(fig.data) == 1
   assert fig.data[0].type == "bar"
   assert list(fig.data[0].x) == expected_trade_dates
   assert len(fig.data[0].y) <= 20
   ```

## Expected Result

- 图表为 Plotly Bar
- x 轴为日期序列
- y 值为每日主力净流入汇总
