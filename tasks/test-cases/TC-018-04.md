## Objective

验证 Tab 3 图表构建函数在 `macd` 由上向下穿越 `macd_signal` 的日期添加红色圆形 marker。

## Preconditions

- `app.components.tab3_stock.build_stock_depth_figure` 或等价图表构建函数可 import
- 准备 DataFrame，其中某日满足 `macd < macd_signal` 且前一日 `macd >= macd_signal`

## Steps

1. 构造含完整技术指标列的 DataFrame。
2. 调用 `fig = build_stock_depth_figure(df)`。
3. 在 `fig.data` 中找到 `name == "死叉"` 的 trace。
4. 断言：
   ```python
   assert death_trace.marker.color == "#E84040"
   assert death_trace.marker.symbol == "circle"
   assert expected_cross_date in list(death_trace.x)
   ```

## Expected Result

- 死叉 trace 存在
- marker 为红色圆形
- marker 日期等于 MACD 下穿信号线的日期
