## Objective

验证 `INTRADAY_FAST` 指标在交易时段内且距离上次成功采集超过
`poll_interval_s` 时被调度。

## Preconditions

- 使用 `MacroStorage(":memory:")`
- NVDA 配置为 `INTRADAY_FAST`，`poll_interval_s=300`
- `storage.get_latest_indicator("NVDA")` 返回 `fetched_at` 为当前模拟时间 301 秒前

## Steps

1. 构造 scheduler。
2. 调用 `get_due_indicators(now=2026-05-15 10:00:00 America/New_York)`。
3. 在返回列表中按 `indicator_id` 查找 NVDA。

## Expected Result

- 返回列表包含 `indicator_id == "NVDA"` 的 `IndicatorConfig`
- 该配置的 `update_class == UpdateClass.INTRADAY_FAST`
