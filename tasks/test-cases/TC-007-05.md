## Objective

验证 `MacroStorage.upsert_fomc()` 持久化 FOMC 记录后，
`get_fomc_upcoming(from_date, lookahead_days)` 只返回闭区间
`[from_date, from_date + lookahead_days]` 内的会议，并按 `meeting_date ASC` 排序。

## Preconditions

- 使用 `MacroStorage(db_path=":memory:")`
- `from_date = date(2026, 5, 17)`
- `lookahead_days = 90`

## Fixture

插入 4 条 FOMC 记录：

| meeting_date | 说明 |
| --- | --- |
| 2026-05-16 | 窗口前，应排除 |
| 2026-05-17 | from_date 当天，应包含 |
| 2026-07-29 | 窗口内，应包含 |
| 2026-08-16 | `from_date + 91d`，应排除 |

记录字段包含 `meeting_type="scheduled"`、`decision_date=None`、`source="test"`。

## Steps

1. 构造 `storage = MacroStorage(":memory:")`
2. 调用 `storage.upsert_fomc(fomc_records)`
3. 调用 `items = storage.get_fomc_upcoming(date(2026, 5, 17), 90)`

## Expected Result

- 返回 2 条记录
- `items[0]["meeting_date"] == date(2026, 5, 17)`
- `items[1]["meeting_date"] == date(2026, 7, 29)`
- 返回列表按 `meeting_date` 升序排列
- 不包含 `2026-05-16` 和 `2026-08-16`
