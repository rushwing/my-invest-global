## Objective

验证 `load_latest_for_ui(db_path)` 在 mock `load_latest_session` 返回含 2 个 signals 的 `AnalysisState` 时，返回的 dict 包含 `signals`, `reasoning`, `captured_at`, `session_id`, `errors`, `change_pct_snapshot` key。

## Preconditions

- `app.pages.analysis.load_latest_for_ui` 可 import
- mock `engine.agent.session_store.load_latest_session` 返回含 2 个 `StockSignal` 的 state

## Steps

1. Mock `load_latest_session` 返回 `AnalysisState`：
   ```python
   state: AnalysisState = {
       "snapshot": FrozenMarketSnapshot(
           session_id="ui-test-001",
           captured_at=datetime(2026, 5, 18, 8, 0, 0),
           ...
       ),
       "signals": [StockSignal(...), StockSignal(...)],
       "reasoning": {"300308": "bullish"},
       "errors": [],
       ...
   }
   ```
2. 调用 `result = load_latest_for_ui(":memory:")`

## Expected Result

- `result` 不为 `None`
- `result["session_id"]` == `"ui-test-001"`
- `result["captured_at"]` == `datetime(2026, 5, 18, 8, 0, 0)`
- `len(result["signals"])` == 2
- `result["reasoning"]["300308"]` == `"bullish"`
- `result["errors"]` == `[]`
- `result["change_pct_snapshot"]` 为 dict（可能为空 `{}`）
- 当 `load_latest_session` 返回 `None` 时，`load_latest_for_ui` 返回 `None`

## Notes

- `load_latest_for_ui` 是 `load_latest_session` 的薄封装，提取 UI 需要的字段
- 纯函数，测试不依赖 Streamlit
- `change_pct_snapshot` 从 `state["snapshot"].change_pct_snapshot` 提取
