## Objective

验证 `render_chip_panel("688143", 131.60, mock_analysis_above_90, mock_summary)` 不抛异常，并在 `above_90_band=True` 时调用 `st.error`。

## Preconditions

- `app.pages.chip_panel.render_chip_panel` 可 import
- mock `streamlit` 的 `st.metric`、`st.error`、`st.info`、`st.plotly_chart` 等调用
- 准备 `mock_analysis_above_90` 和含 `bars` 的 `mock_summary`

## Steps

1. monkeypatch `app.pages.chip_panel.st` 为 mock 对象。
2. 调用 `render_chip_panel("688143", 131.60, mock_analysis_above_90, mock_summary)`。
3. 断言：
   ```python
   st.error.assert_called_once()
   assert st.warning.call_count == 0
   ```

## Expected Result

- 函数不抛异常
- `st.error` 被调用恰好 1 次
- 图表渲染和指标卡片渲染路径均可执行

## Notes

- 不启动 Streamlit server
