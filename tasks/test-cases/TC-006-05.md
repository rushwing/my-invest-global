## Objective

验证 `fetch_capex_quarterly()` 对 10-Q Q2 累计值执行正确的单季反算。

## Preconditions

- Mock 返回 MSFT 2025 fixture：Q1=$3B (fp=Q1, form=10-Q)，Q2_cum=$7B (fp=Q2, form=10-Q)
- 两条记录均属同一财年 (fy=2025)

## Fixture JSON (companyfacts 格式)

```json
{
  "facts": {
    "us-gaap": {
      "PaymentsToAcquirePropertyPlantAndEquipment": {
        "units": {
          "USD": [
            {"end": "2025-03-31", "val": 3000000000, "form": "10-Q", "fp": "Q1", "fy": 2025},
            {"end": "2025-06-30", "val": 7000000000, "form": "10-Q", "fp": "Q2", "fy": 2025}
          ]
        }
      }
    }
  }
}
```

## Steps

1. Mock `_get` 返回上述 fixture
2. `records = src.fetch_capex_quarterly("0000789019")`
3. 取 `fp == "Q2"` 的记录

## Expected Result

- `q2_record["capex_usd"]` == `4.0e9`（即 4B USD，Q2_cum 7B - Q1 3B）
- `q2_record["filing_form"]` == `"10-Q"`
- `q2_record["fiscal_quarter"]` == `"2025Q2"`（或等价格式）
