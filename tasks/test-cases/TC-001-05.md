## Preconditions

- `data/cache/macro_state.json` 不存在

## Steps

1. `set_macro_state_override(MacroState.YELLOW)`
2. 读取 `data/cache/macro_state.json` 内容 → 记为 `content1`
3. `set_macro_state_override(MacroState.YELLOW)`（再次相同值）
4. 读取 `data/cache/macro_state.json` 内容 → 记为 `content2`

## Expected Result

- 两次调用均不抛出异常
- `content1 == content2 == '{"state": "yellow"}'`
- `get_macro_state() == MacroState.YELLOW`
