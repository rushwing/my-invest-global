## Objective

验证 `retrieve(query, db, mock_embedder)` 能从已写入的 LanceDB 表召回结果，并返回
`RetrieveResult` 列表。

## Preconditions

- 使用 memory LanceDB
- 已通过 `ingest_pdf()` 写入至少 3 行 chunk
- mock embedder 对 query 返回 1024 维向量

## Steps

1. 调用 `retrieve("液冷散热收入", db, mock_embedder, top_k_ann=3, top_k_final=2)`。
2. 检查返回列表和字段。

## Expected Result

- 返回值是 `list[RetrieveResult]`
- 返回数量至少为 1
- 每个 `chunk_id` 匹配正则 `r"^.+_\\d{4}$"`
- 每个结果的 `text` 非空
