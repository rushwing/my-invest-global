## Objective

验证 `ReleaseCalendar.is_release_day()` 能驱动 `MONTHLY_FIXED` 指标准确触发。

## Preconditions

- 使用 `MacroStorage(":memory:")`
- `release_dates` 中预填充 `indicator_id="CPIAUCSL"`，`release_date=2026-05-13`

## Steps

1. 调用 `release_calendar.is_release_day("CPIAUCSL", date(2026, 5, 13))`。
2. 调用 `release_calendar.is_release_day("CPIAUCSL", date(2026, 5, 14))`。
3. 在 `2026-05-13` 调用 scheduler。

## Expected Result

- 第一次 `is_release_day()` 返回 True
- 第二次 `is_release_day()` 返回 False
- `get_due_indicators()` 在发布日返回 CPIAUCSL 配置，非发布日不返回
