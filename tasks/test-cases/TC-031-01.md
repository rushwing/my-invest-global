## Objective

验证 `format_signals_df([signal_with_code_300308], {"300308": 2.3})` 返回的 DataFrame 包含 `code`, `name`, `composite_score`, `action_code`, `change_pct` 列，且 `change_pct` 值为 `"+2.30%"`。

## Preconditions

- `app.pages.analysis.format_signals_df` 可 import
- `pandas` 已安装
- 构造 `StockSignal(code="300308", ...)` 含 `action_code="hold_add"`, `composite_score=78.5`

## Steps

1. 构造 `StockSignal`：
   ```python
   sig = StockSignal(
       code="300308", name="中际旭创", category="白马股",
       technical_score=80, fundamental_score=78, sentiment_score=77,
       composite_score=78.5, action="持有加仓", action_code="hold_add",
       reasoning="测试", signals={}
   )
   ```
2. 调用 `df = format_signals_df([sig], change_pct_snapshot={"300308": 2.3})`

## Expected Result

- `df` 类型为 `pd.DataFrame`
- `set(df.columns)` `⊇` `{"code", "name", "composite_score", "action_code", "change_pct"}`
- `df.iloc[0]["code"]` == `"300308"`
- `df.iloc[0]["composite_score"]` == `78.5`
- `df.iloc[0]["change_pct"]` == `"+2.30%"`（带正负号和小数点后两位）
- `df.iloc[0]["action_code"]` 包含中文标签（如 `🟢 持有加仓`）
- 当 `change_pct_snapshot=None` 时，`change_pct` == `"+0.00%"`

## Notes

- `format_signals_df` 是纯函数，不依赖 Streamlit
- `composite_score` 通过 `round(s.composite_score, 1)` 保留一位小数
- `change_pct` 通过 `f"{pct:+.2f}%"` 格式化
