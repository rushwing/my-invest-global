## Preconditions

- `engine/macro_gate` 可正常 import

## Steps

1. 检查 `MacroState.GREEN == "green"`
2. 检查 `MacroState.YELLOW == "yellow"`
3. 检查 `MacroState.RED == "red"`
4. 检查 `MacroState.GREEN.value == "green"`
5. 检查 `isinstance(MacroState.GREEN, str)` 为 True
6. 调用 `check_portfolio_balance(holdings_df, macro_state=MacroState.GREEN.value)` — 传入 `.value` 字符串

## Expected Result

- Steps 1–5 全部为 True（无异常）
- Step 6：函数正常执行，不抛 ValueError（`.value` 是合法的 `"green"` 字符串）

## Notes

mypy --strict 不接受直接传入 `MacroState.GREEN`（类型为 `MacroState`，非 `Literal[...]`）；
正确调用方式是 `.value`，REQ-002 更新签名后可去掉。
此 TC 验证 runtime 行为，mypy 静态检查由 TC-001-07 间接覆盖（`uv run mypy`）。
