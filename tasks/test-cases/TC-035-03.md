## Objective

验证 `app.pages.backtest.format_backtest_result(result)` 是纯函数，并将 `BacktestResult` 转换为包含四个核心指标列的单行 DataFrame。

## Preconditions

- `app.pages.backtest.format_backtest_result` 可 import
- `engine.backtest.runner.BacktestResult` 可 import
- 准备一个手工构造的 `BacktestResult`

## Steps

1. 构造 `BacktestResult`：
   ```python
   import pandas as pd
   from engine.backtest.runner import BacktestResult

   result = BacktestResult(
       code="300308",
       strategy="ma_cross",
       total_return=0.1234,
       benchmark_return=0.2345,
       max_drawdown=-0.0567,
       win_rate=0.5,
       num_trades=4,
       equity_curve=pd.Series([1.0, 1.05, 1.12], index=pd.date_range("2024-01-01", periods=3)),
   )
   ```
2. 调用：
   ```python
   df = format_backtest_result(result)
   ```
3. 验证 DataFrame 结构：
   ```python
   assert isinstance(df, pd.DataFrame)
   assert len(df) == 1
   for col in ["total_return", "benchmark_return", "max_drawdown", "win_rate"]:
       assert col in df.columns
       assert isinstance(df.loc[0, col], float)
   ```
4. 验证函数不修改原始 result：
   ```python
   assert result.total_return == 0.1234
   assert result.equity_curve.iloc[0] == 1.0
   ```

## Expected Result

- 返回值是单行 `pd.DataFrame`
- DataFrame 包含 `total_return` / `benchmark_return` / `max_drawdown` / `win_rate` 四列
- 四列值均为 float
- 函数不依赖 Streamlit、不访问网络、不修改输入对象
