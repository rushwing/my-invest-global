## Objective

验证 ScarcityMatrix chip 底部 5 日资金流 mini strip 对 `[+1, -2, +3, -1, +5]` 依次渲染绿红绿红绿。

## Preconditions

- `app.components.tab4_signals.render_flow_strip` 或等价函数可 import
- 可捕获生成的 HTML

## Steps

1. 构造 5 行 fund_flow 数据，`main_net_inflow=[1, -2, 3, -1, 5]`，日期左旧右新。
2. 调用 `html = render_flow_strip(flow_df)`。
3. 断言 HTML 中 5 个 `.flow-cell` 的 class 顺序为：
   ```python
   ["pos", "neg", "pos", "neg", "pos"]
   ```

## Expected Result

- 资金流格子数量为 5
- 颜色顺序与正负号一致

## Notes

- 正数绿色 `#00C47A`，负数红色 `#E84040`
