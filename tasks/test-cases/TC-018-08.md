## Objective

验证日期范围选择 `1M` 时，图表 x 轴数据被截断为最近 30 天。

## Preconditions

- `app.components.tab3_stock.apply_date_range` 或等价过滤 helper 可 import
- 准备 180 行按 date 升序排列的 DataFrame

## Steps

1. 调用 `filtered = apply_date_range(df, "1M")`。
2. 断言：
   ```python
   assert len(filtered) <= 30
   assert filtered["date"].min() >= df["date"].max() - pd.Timedelta(days=30)
   assert filtered["date"].max() == df["date"].max()
   ```

## Expected Result

- `1M` 选择只保留最近 30 天窗口
- 最大日期保持为原始数据最新日期

## Notes

- 若实现无 helper，可通过 `render_tab3()` 捕获 Figure x 轴数据测试
