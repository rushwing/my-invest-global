## Objective

验证 `run_backtest(code="300308", strategy="macd_cross", df=mock_ohlcv_df)` 支持 MACD 金叉/死叉策略，并返回合法的交易次数与胜率范围。

## Preconditions

- `engine.backtest.runner.run_backtest` 和 `BacktestResult` 可 import
- mock OHLCV 至少 60 行，包含小写 `open/high/low/close/volume`
- `close` 序列先下跌再上涨再回落，使 MACD DIF/DEA 有机会发生交叉

## Steps

1. 构造 mock OHLCV：
   ```python
   import numpy as np
   import pandas as pd

   dates = pd.date_range("2024-01-01", periods=70, freq="D")
   close = np.r_[
       np.linspace(120, 90, 25),
       np.linspace(91, 135, 30),
       np.linspace(134, 110, 15),
   ]
   df = pd.DataFrame({
       "open": close - 0.5,
       "high": close + 1.0,
       "low": close - 1.0,
       "close": close,
       "volume": [1_000_000] * len(close),
   }, index=dates)
   ```
2. 调用：
   ```python
   result = run_backtest(code="300308", strategy="macd_cross", df=df)
   ```
3. 验证返回结构和策略字段：
   ```python
   assert isinstance(result, BacktestResult)
   assert result.code == "300308"
   assert result.strategy == "macd_cross"
   assert isinstance(result.total_return, float)
   assert isinstance(result.benchmark_return, float)
   assert isinstance(result.max_drawdown, float)
   assert isinstance(result.win_rate, float)
   assert isinstance(result.num_trades, int)
   assert isinstance(result.equity_curve, pd.Series)
   ```
4. 验证交易统计范围：
   ```python
   assert result.num_trades >= 0
   assert 0.0 <= result.win_rate <= 1.0
   assert len(result.equity_curve) == len(df)
   assert result.equity_curve.index.equals(df.index)
   ```
5. 可选地直接验证策略信号函数：
   ```python
   from engine.backtest.strategies import macd_cross_signals

   signals = macd_cross_signals(df)
   assert isinstance(signals, pd.Series)
   assert len(signals) == len(df)
   assert set(signals.dropna().unique()).issubset({0, 1})
   ```

## Expected Result

- `macd_cross` 返回 `BacktestResult`
- `num_trades` 为非负整数
- `win_rate` 在 `[0.0, 1.0]` 区间
- `equity_curve` 与输入 df 同长度、同 index
- MACD 策略信号只包含 `0/1`

## Notes

- 此 TC 不要求固定收益率数值，避免过度绑定 EMA 细节
- 重点验证 MACD 策略入口存在、结果结构正确、交易统计口径合法
