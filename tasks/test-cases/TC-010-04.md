## Objective

验证中国市场关键指标中的 SGX A50 期货可以通过 AKShare 宏观 source 写入。

## Preconditions

- `AKShareMacroSource.fetch_sgx_a50()` mock 返回一条有效记录
- scheduler 返回 `indicator_id="SGX_A50"` 的 due 配置
- 使用 `MacroStorage(":memory:")`

## Steps

1. 调用 `run_once(groups=["N"])`。
2. 查询 `macro_indicators`。

## Expected Result

- 存在 `indicator_id == "SGX_A50"` 的行
- `group_code == "N"`
- `value` 非 None
- `source == "akshare_macro"`
