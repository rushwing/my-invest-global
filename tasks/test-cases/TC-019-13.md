## Objective

验证 MacroCyclePanel 顶部右图使用 `load_market_breadth()` 返回的 `up_ratio` 生成折线图，且 y 值保持在 `[0, 1]`。

## Preconditions

- `app.components.tab4_signals.build_market_breadth_chart` 或等价函数可 import
- 准备 20 行 `breadth_df`，列含 `date/up_ratio`

## Steps

1. 调用 `fig = build_market_breadth_chart(breadth_df)`。
2. 断言：
   ```python
   assert len(fig.data) >= 1
   assert fig.data[0].type == "scatter"
   assert min(fig.data[0].y) >= 0
   assert max(fig.data[0].y) <= 1
   ```

## Expected Result

- 图表为 Plotly Scatter 折线
- y 轴数值均在 `[0, 1]`
- 函数不抛异常

## Notes

- 0.5 基准线可作为附加断言
