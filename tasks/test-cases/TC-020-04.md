## Objective

验证 `data/processed/` 无 signals、recommendations、scarcity、rebalance history 文件时，4 个 Tab 渲染友好占位符，不抛 `KeyError` 或 `FileNotFoundError`。

## Preconditions

- `render_tab1`, `render_tab2`, `render_tab3`, `render_tab4` 可 import
- monkeypatch 数据目录为空，或 monkeypatch loader 返回 `None`/空 DataFrame
- monkeypatch Streamlit 渲染函数

## Steps

1. 让所有 loader 返回缺文件场景：`None`、空 dict 或空 DataFrame。
2. 分别调用 4 个 Tab render 函数。
3. 断言调用均不抛异常。
4. 断言至少调用一次 `st.info`、`st.warning` 或占位符渲染函数。

## Expected Result

- 无数据文件时 UI 走降级占位路径
- 不抛 `KeyError` / `FileNotFoundError`

- 可拆成多个实现级单元测试，但同属此 TC 覆盖
