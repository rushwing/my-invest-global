## Objective

验证 `parse_chip_screenshot("tests/fixtures/chip_688143_20260519.png")` 在 mock Claude API 返回标准 JSON 时，返回 `ChipAnalysis`，并对 `range_70` 空值应用 fallback。

## Preconditions

- `engine.agent.chip_screenshot_parser.parse_chip_screenshot` 可 import
- fixture `tests/fixtures/chip_688143_20260519.png` 存在
- mock `anthropic.Anthropic().messages.create`，不发真实 API 请求

## Steps

1. mock Claude API 返回 JSON 字符串：
   ```json
   {"code":"688143","current_price":131.60,"avg_cost":99.96,"profitable_pct":0.9973,"concentration":33.92,"range_70_lower":null,"range_70_upper":null,"range_90_lower":63.28,"range_90_upper":128.24}
   ```
2. 调用 `analysis = parse_chip_screenshot("tests/fixtures/chip_688143_20260519.png")`。
3. 断言：
   ```python
   assert abs(analysis.avg_cost - 99.96) < 0.01
   assert analysis.profitable_pct > 0.99
   assert analysis.range_70_lower == analysis.range_90_lower
   assert analysis.range_70_upper == analysis.range_90_upper
   ```

## Expected Result

- 返回 `ChipAnalysis`
- 不发真实 Claude 请求
- `range_70` fallback 到 `range_90`

## Notes

- fixture 只用于测试图片读取和 base64 编码路径
