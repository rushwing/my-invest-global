## Objective

验证 `compute_indicators()` 输出的 `bb_upper/bb_mid/bb_lower` 在所有非空行严格满足 `upper > mid > lower`。

## Preconditions

- `app.data_loader.compute_indicators` 可 import
- 准备 180 行有波动的 OHLCV DataFrame，`close` 不为常数

## Steps

1. 调用 `df = compute_indicators(ohlcv_df)`。
2. 过滤 `bands = df.dropna(subset=["bb_upper", "bb_mid", "bb_lower"])`。
3. 断言：
   ```python
   assert len(bands) >= 160
   assert (bands["bb_upper"] > bands["bb_mid"]).all()
   assert (bands["bb_mid"] > bands["bb_lower"]).all()
   ```

## Expected Result

- Bollinger Bands 三列存在
- 第 20 日之后的非空 band 行均严格有序

## Notes

- 输入价格序列需包含非零标准差，避免上下轨退化相等
