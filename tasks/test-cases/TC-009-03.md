## Objective

验证 orchestrator 尊重 scheduler 的时间窗口过滤，不绕过调度器直接采集 NVDA。

## Preconditions

- 使用 mock scheduler 或真实 scheduler，模拟 `2026-05-15 01:00 America/New_York`
- Group K 包含 NVDA `INTRADAY_FAST` 配置
- Yahoo source 为 mock，可记录是否被调用

## Steps

1. 调用 `run_once(groups=["K"])`。
2. 检查 Yahoo source 的 NVDA fetch 调用。

## Expected Result

- NVDA 不触发采集
- Yahoo source 不因 NVDA 被调用
- `run_once()` 正常返回，不抛异常
