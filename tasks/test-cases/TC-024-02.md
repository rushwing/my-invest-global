## Objective

验证 mock LLM 在生成 Cypher 阶段抛异常时，`KGQuerier.query()` 捕获异常并返回
`KGQueryResult.error`，不向上传播。

## Preconditions

- 注入 mock `llm`
- mock LLM 的 `invoke()` 抛 `ValueError("timeout")`
- 注入 `session_runner` mock

## Steps

1. 调用 `KGQuerier(conn).query("问题", llm=mock_llm, schema="schema", session_runner=runner)`。

## Expected Result

- `result.error == "timeout"`
- `result.answer == ""`
- `result.raw_results == []`
- `session_runner` 未被调用
- 测试过程中无异常传播
