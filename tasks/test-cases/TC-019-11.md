## Objective

验证点击某 chip 的 `"进入 →"` 按钮后，`st.session_state["tab3_code"]` 被设为该股票 code，并调用 `st.rerun()`。

## Preconditions

- `app.components.tab4_signals.render_scarcity_chip` 或等价函数可 import
- monkeypatch `st.button` 对 `key="enter_300308"` 返回 `True`
- monkeypatch `st.session_state` 与 `st.rerun`

## Steps

1. 渲染 code 为 `"300308"` 的 chip。
2. 断言：
   ```python
   assert st.session_state["tab3_code"] == "300308"
   st.rerun.assert_called_once()
   ```

## Expected Result

- CTA 正确写入 Tab 3 预选 code
- 触发 Streamlit rerun

- 与 TC-018-09/10 组成跨 Tab 导航闭环
