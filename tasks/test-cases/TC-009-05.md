## Objective

验证 10Y-2Y 利差在 storage upsert 后由本地逻辑生成。

## Preconditions

- 使用 `MacroStorage(":memory:")`
- FRED source mock 返回 DGS10=4.5、DGS2=4.1，日期相同
- scheduler 返回 DGS10 和 DGS2 的 due 配置

## Steps

1. 调用 `run_once(groups=["K"])`。
2. 查询 `macro_indicators` 中 `indicator_id="T10Y2Y"` 的最新记录。

## Expected Result

- 存在 `T10Y2Y` 行
- `value == 0.4`，允许浮点误差 0.0001
- `period_date` 与 DGS10/DGS2 输入日期一致
- `source` 标识为本地计算来源
