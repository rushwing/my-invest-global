## Objective

验证自动计算路径能写入或覆盖 `macro_state.json`。

## Preconditions

- 使用临时 `data/cache`
- 场景 A：`macro_state.json` 不存在
- 场景 B：`macro_state.json` 存在且含 `"auto_computed": true`

## Steps

1. 调用 `write_cache({"composite_state": "yellow", ...})`。
2. 读取 `macro_state.json`。
3. 对场景 B 重复一次，写入 `"red"`。

## Expected Result

- 文件被创建或更新
- JSON 中 `state` 等于传入的 `composite_state`
- JSON 中包含 `auto_computed: true`、`computed_at`、`capex_as_of`
