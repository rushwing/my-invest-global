## Objective

验证 `load_fund_flow_5d([], days=5)` 返回空 DataFrame，并且不尝试连接 DuckDB。

## Preconditions

- `app.data_loader.load_fund_flow_5d` 可 import
- monkeypatch `duckdb.connect` 为 spy

## Steps

1. 调用 `df = load_fund_flow_5d([], days=5)`。
2. 断言：
   ```python
   assert list(df.columns) == ["code", "trade_date", "main_net_inflow"]
   assert df.empty
   duckdb.connect.assert_not_called()
   ```

## Expected Result

- 空 code 输入快速返回稳定空 schema
- 不访问 DuckDB
