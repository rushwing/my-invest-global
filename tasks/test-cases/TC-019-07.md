## Objective

验证 ScarcityMatrix 渲染 `owned=True` 的股票时，在 chip 标题前显示 `◉`。

## Preconditions

- `app.components.tab4_signals.render_scarcity_chip` 或等价函数可 import
- 可 monkeypatch `st.html` 或 `st.markdown` 捕获 chip HTML

## Steps

1. 构造一行 scarcity 数据：`owned=True`, `name="中际旭创"`, `code="300308"`。
2. 渲染 chip。
3. 断言捕获 HTML 包含：
   ```python
   assert "◉中际旭创" in html or "◉ 中际旭创" in html
   ```

## Expected Result

- 当前持有股票以 `◉` 标记

## Notes

- `warn=True` 的 `⚠` 标记可在实现中补充相邻断言
