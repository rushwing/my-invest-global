## Objective

验证 `list_sessions(":memory:", limit=2)` 在插入 3 个 session（captured_at 递增）时返回 2 条，且第一条是最新的。

## Preconditions

- `engine.agent.session_store.save_session`, `list_sessions` 可 import
- DuckDB `:memory:` 空表

## Steps

1. 构造 3 个 `AnalysisState`，`captured_at` 依次为：
   - `2026-05-16 08:00:00`
   - `2026-05-17 08:00:00`
   - `2026-05-18 08:00:00`
2. 依次调用 `save_session(":memory:", s)`（3 次，按时间顺序）
3. 调用 `rows = list_sessions(":memory:", limit=2)`

## Expected Result

- `len(rows)` == 2
- `rows[0]["captured_at"]` >= `rows[1]["captured_at"]`（降序）
- `rows[0]["session_id"]` 为最近一次 save 的 session_id
- 每条记录包含 `session_id`, `captured_at`, `prompt_hash` 三个 key

## Notes

- `list_sessions` 仅返回元数据（不返回完整 state/signals），减少数据传输量
- `limit` 参数设上限避免全表扫描（默认 30）
