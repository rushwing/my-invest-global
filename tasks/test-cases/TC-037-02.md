## Objective

验证当前价格低于 `range_90_lower` 时，`below_90_band=True` 且摘要包含“套牢”。

## Preconditions

- `analyze_chip` 可 import
- 准备 `mock_summary_below_band`，其中 `range_90_lower > current_price`

## Steps

1. 构造 `mock_summary_below_band`，例如 `range_90_lower=15.0`、`range_90_upper=20.0`、`avg_cost=16.0`。
2. 调用 `analysis = analyze_chip("000001", 12.0, mock_summary_below_band)`。
3. 断言：
   ```python
   assert analysis.below_90_band is True
   assert analysis.above_90_band is False
   assert "套牢" in analysis.signal_summary
   assert len(analysis.signal_summary) <= 80
   ```

## Expected Result

- `below_90_band=True`
- `signal_summary` 命中套牢模板

## Notes

- 不依赖具体股票真实数据
