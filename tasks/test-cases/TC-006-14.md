## Objective

验证 `_df_to_records()` 返回的记录具有与 `_make_period_record()` 一致的
schema 字段，确保 AKShare 路径兼容 REQ-007 upsert。

## Fixture

```python
import pandas as pd
df = pd.DataFrame([{"日期": "2026-03-01", "今值": "2.1"}])
```

## Steps

1. 构造 `AKShareMacroSource(rate_limiter)`
2. 调用 `src._df_to_records(df, indicator_id="CN_CPI", group_code="I",
   date_col="日期", value_col="今值", market_tz="Asia/Shanghai",
   value_unit="percent_yoy", start_date=None)`

## Expected Result

- 返回 1 条记录
- `rec["indicator_id"]` == `"CN_CPI"`
- `rec["value"]` == `2.1`
- `rec` 包含 `realtime_start`, `vintage_as_of`, `utc_ts` 等 schema 字段
