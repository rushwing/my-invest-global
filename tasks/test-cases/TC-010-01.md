## Objective

验证 CapEx 入库使用单季口径，而不是 10-Q 累计口径。

## Preconditions

- 使用 MSFT SEC EDGAR fixture：Q1=3B，Q2_cum=7B
- 使用 `MacroStorage(":memory:")`

## Steps

1. mock `SECEdgarSource.fetch_capex_quarterly("0000789019")` 返回反算前 fixture。
2. 调用 `run_once(groups=["L"])` 或直接调用 CapEx 处理路径。
3. 查询 `get_capex_quarters("MSFT", 4)`。

## Expected Result

- Q2 记录存在
- Q2 `capex_usd == 4.0` billion
- `filing_form == "10-Q"`
