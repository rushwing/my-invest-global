## Objective

验证 Gauge 对 `abs(deviation)=6%` 使用警告颜色 `#F5A623`，但不将触发器状态设为 triggered。

## Preconditions

- `app.components.tab4_signals.gauge_color_for_deviation` 或等价 helper 可 import
- `compute_trigger_status()` 或触发状态 helper 可 import

## Steps

1. 调用 `gauge_color_for_deviation(6)`。
2. 构造弹性仓位比目标偏离 6% 的 holdings DataFrame。
3. 调用触发状态计算。
4. 断言：
   ```python
   assert gauge_color == "#F5A623"
   assert "triggered" not in set(triggers["status"])
   ```

## Expected Result

- Gauge 为 neutral 黄色
- ±10% 触发条件未触发

## Notes

- 明确区分 ±5% 警告带与 ±10% 触发器
