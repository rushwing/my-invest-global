## Objective

验证 `fetch_series()` 在调用方未传 `realtime_start` 参数时，
仍能从 FRED payload 的 observation-level `realtime_start` 字段捕获
vintage 日期，而非记录 `None`。

## Steps

1. Mock `_get` 返回：
   ```json
   {
     "observations": [
       {"date": "2026-04-01", "value": "3.5", "realtime_start": "2026-04-11"}
     ]
   }
   ```
2. 调用 `fred.fetch_series("CPIAUCSL")`（不传 `realtime_start` 参数）

## Expected Result

- `result[0]["realtime_start"]` == `date(2026, 4, 11)`（来自 payload）
- 而非 `None`
