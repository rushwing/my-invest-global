## Objective

验证 `_make_period_record()` 返回的 dict 包含 `macro_indicators` 表所需的
全部列名，避免 REQ-007 upsert 时列名不匹配导致 KeyError 或 NULL 列。

## Required Keys

indicator_id, group_code, period_date, market_tz, value, value_unit, value_raw,
source, realtime_start, vintage_as_of, utc_ts, fetched_at

## Steps

1. 实例化任意 `MacroAbstractSource` 子类（如 `AKShareMacroSource`）
2. 调用 `src._make_period_record(indicator_id="X", group_code="I",
   period_date=date.today(), market_tz="UTC", value=1.0)`
3. 取返回 dict 的 key 集合

## Expected Result

- 返回 dict 的 keys 是上述必需字段的超集（即每个必需字段都存在）
