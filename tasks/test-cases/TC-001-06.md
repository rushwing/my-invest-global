## Preconditions

- 模块可通过 `python -m engine.macro_gate` 运行

## Steps

1. 运行 `python -m engine.macro_gate --set foo`（无效值）
2. 运行 `python -m engine.macro_gate --set GREEN`（大写，不合法）
3. 运行 `python -m engine.macro_gate`（缺少 `--set` 参数）

## Expected Result

- Step 1–2：进程以非零退出码退出（`SystemExit` 或 `sys.exit(1)`）；stderr 输出包含可读错误信息
- Step 3：进程以非零退出码退出；stderr 提示缺少参数

## Variant

合法调用验证：`python -m engine.macro_gate --set green` → exit code 0；`get_macro_state() == MacroState.GREEN`。
