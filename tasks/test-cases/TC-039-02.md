## Objective

验证 `build_chip_chart(mock_summary, current_price=131.60)` 返回 Plotly `go.Figure`，并按档位中点相对现价生成正确颜色和标注线。

## Preconditions

- `app.pages.chip_panel.build_chip_chart` 可 import
- `plotly.graph_objects as go` 可 import
- `mock_summary.bars` 至少包含一个中点低于现价、一个中点高于或等于现价的 bar

## Steps

1. 构造 `mock_summary`，包含 `avg_cost`、`range_90_lower`、`range_90_upper` 和 `bars`。
2. 调用 `fig = build_chip_chart(mock_summary, current_price=131.60)`。
3. 断言：
   ```python
   assert isinstance(fig, go.Figure)
   colors = list(fig.data[0].marker.color)
   assert "#00C47A" in colors
   assert "#E84040" in colors
   labels = str(fig.layout.shapes) + str(fig.layout.annotations)
   assert "现价" in labels
   assert "均成本" in labels
   ```
4. 对每个 bar 计算中点，断言低于现价为绿，高于或等于现价为红。

## Expected Result

- 返回 `go.Figure`
- 颜色规则符合 REQ-039
- 标注包含现价和均成本
