## Objective

验证相同输入连续调用 `analyze_chip` 两次，返回相同结果，无随机性、无外部依赖、无副作用。

## Preconditions

- `analyze_chip` 可 import
- 准备固定 mock `ChipSummary`

## Steps

1. 构造固定 summary。
2. 调用：
   ```python
   a1 = analyze_chip("688143", 131.60, summary)
   a2 = analyze_chip("688143", 131.60, summary)
   ```
3. 断言：
   ```python
   assert a1.model_dump() == a2.model_dump()
   ```
4. 使用 monkeypatch 禁止数据库或网络函数被调用（如有相关 import），确认 `analyze_chip` 不触发外部访问。

## Expected Result

- 两次返回完全一致
- 不访问 DuckDB
- 不访问网络或 akshare

## Notes

- 该 TC 只约束 `analyze_chip`，不约束 `load_and_analyze`
