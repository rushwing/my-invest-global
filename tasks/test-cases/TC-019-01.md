## Objective

验证 Tab 4 在宏观状态为 `"green"` 时，弹性仓位仪表盘目标为 `38%`。

## Preconditions

- `app.components.tab4_signals.elastic_target_for_macro` 或等价 helper 可 import
- 若无 helper，可 monkeypatch `load_macro_state()` 返回 `{"state": "green"}`

## Steps

1. 调用目标计算 helper，或渲染 Gauge 并捕获目标文本。
2. 断言：
   ```python
   assert target == 38
   ```

## Expected Result

- `green` 宏观状态映射到弹性仓位目标 `38%`

## Notes

- 不依赖真实 `data/cache/macro_state.json`
