## Preconditions

- `data/cache/macro_state.json` 不存在（或已删除）
- `engine/macro_gate` 可正常 import

## Steps

1. `set_macro_state_override(MacroState.GREEN)`
2. `result1 = get_macro_state()`
3. `set_macro_state_override(MacroState.RED)`
4. `result2 = get_macro_state()`
5. `set_macro_state_override(MacroState.YELLOW)`
6. `result3 = get_macro_state()`

## Expected Result

- `result1 == MacroState.GREEN`
- `result2 == MacroState.RED`
- `result3 == MacroState.YELLOW`
- 每次 set 后，`data/cache/macro_state.json` 内容为 `{"state": "<lowercase_value>"}`

## Postconditions

无需清理（测试框架使用临时目录 fixture，隔离 `data/cache/`）
