## Objective

验证 DuckDB 中不存在 `stock_quotes` 表时，`load_market_breadth(days=20)` 返回稳定空 DataFrame，不抛异常。

## Preconditions

- `app.data_loader.load_market_breadth` 可 import
- 使用不含 `stock_quotes` 表的临时 DuckDB

## Steps

1. monkeypatch `DB_PATH` 指向空 DuckDB。
2. 调用 `df = load_market_breadth(days=20)`。
3. 断言：
   ```python
   assert list(df.columns) == ["date", "up_ratio"]
   assert df.empty
   ```

## Expected Result

- 缺表不抛异常
- 返回稳定空 schema

## Notes

- 覆盖本地首次运行无行情快照表场景
