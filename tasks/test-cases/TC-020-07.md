## Objective

验证 `load_fund_flow_5d(["300308"], days=5)` 在 DuckDB 有表和数据时返回包含 `code/trade_date/main_net_inflow` 的 DataFrame。

## Preconditions

- `app.data_loader.load_fund_flow_5d` 可 import
- 使用 in-memory DuckDB 或临时 DuckDB 文件创建 `stock_fund_flow`
- 插入至少 5 天 `300308` 数据

## Steps

1. monkeypatch `DB_PATH` 指向临时 DuckDB。
2. 调用 `df = load_fund_flow_5d(["300308"], days=5)`。
3. 断言：
   ```python
   assert list(df.columns) == ["code", "trade_date", "main_net_inflow"]
   assert set(df["code"]) == {"300308"}
   assert len(df) >= 1
   ```

## Expected Result

- 返回 DataFrame 列结构稳定
- 仅包含请求 code 的资金流数据

## Notes

- 可使用当前日期附近的 trade_date，避免 cutoff 过滤掉 fixture
