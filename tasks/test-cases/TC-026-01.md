## Objective

验证 `parse_pdf_to_chunks()` 使用 pymupdf 提取页面文本后能生成非空 chunk，并保持
`chunk_index` 从 0 递增。

## Preconditions

- mock `fitz.open` 返回 3 页对象
- 每页 `get_text()` 返回 300 个以空格分隔的词

## Steps

1. 调用 `parse_pdf_to_chunks(Path("mock.pdf"))`。
2. 检查返回的 `ParsedChunk` 列表。

## Expected Result

- chunk 数量大于等于 1
- 每个 chunk 的 `text` 非空
- `chunk_index` 从 0 开始连续递增
- 每个 chunk 的 `page_start <= page_end`
