## Preconditions

- `engine/macro_gate` 可正常 import
- `uv run mypy` 可执行

## Steps

1. 检查 `TARGET_ELASTIC_BY_MACRO_STATE[MacroState.GREEN] == 0.38`
2. 检查 `TARGET_ELASTIC_BY_MACRO_STATE[MacroState.YELLOW] == 0.33`
3. 检查 `TARGET_ELASTIC_BY_MACRO_STATE[MacroState.RED] == 0.20`
4. 检查 `set(TARGET_ELASTIC_BY_MACRO_STATE.keys()) == {MacroState.GREEN, MacroState.YELLOW, MacroState.RED}`（无缺漏键）
5. 运行 `uv run mypy engine/macro_gate.py --strict`，退出码为 0

## Expected Result

- Steps 1–4：断言全部通过
- Step 5：mypy 输出 `Success: no issues found`，退出码 0
