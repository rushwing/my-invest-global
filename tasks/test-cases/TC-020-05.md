## Objective

验证 `compute_portfolio_metrics(df, "yellow")` 返回 `elastic_target=33`，并正确计算白马/弹性市值与占比。

## Preconditions

- `app.data_loader.compute_portfolio_metrics` 可 import
- 准备包含 `category` 与 `market_value` 的 holdings DataFrame

## Steps

1. 构造白马股市值 67、弹性股市值 33 的 DataFrame。
2. 调用 `metrics = compute_portfolio_metrics(df, "yellow")`。
3. 断言：
   ```python
   assert metrics["elastic_target"] == 33
   assert metrics["total_mv"] == 100
   assert metrics["white_pct"] == pytest.approx(67)
   assert metrics["elastic_pct"] == pytest.approx(33)
   ```

## Expected Result

- yellow 宏观状态目标弹性仓位为 33
- 基础组合指标计算正确

## Notes

- 可额外断言大写 `"YELLOW"` 兼容
