## Objective

验证 ScarcityMatrix chip 对 `composite_score=48` 使用 bear 红色 `#E84040`。

## Preconditions

- `app.components.tab4_signals.chip_color` 或等价 helper 可 import

## Steps

1. 调用 `color = chip_color(48)`。
2. 断言：
   ```python
   assert color == "#E84040"
   ```

## Expected Result

- 分数小于 50 的 chip 使用红色
