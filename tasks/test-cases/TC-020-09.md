## Objective

验证 DuckDB 中不存在 `stock_fund_flow` 表时，`load_fund_flow_5d()` 返回稳定空 DataFrame，不抛异常。

## Preconditions

- `app.data_loader.load_fund_flow_5d` 可 import
- 使用不含 `stock_fund_flow` 表的临时 DuckDB

## Steps

1. monkeypatch `DB_PATH` 指向空 DuckDB。
2. 调用 `df = load_fund_flow_5d(["300308"], days=5)`。
3. 断言：
   ```python
   assert list(df.columns) == ["code", "trade_date", "main_net_inflow"]
   assert df.empty
   ```

## Expected Result

- 缺表不抛异常
- 返回稳定空 schema

## Notes

- 覆盖本地首次运行无资金流表场景
