## Objective

验证 `Embedder` 的 BGE-M3 模型懒加载行为，避免 import 模块时触发 torch 或模型下载。

## Preconditions

- 可 import `engine.rag.embedder.Embedder`
- 未调用 `embed()` 或 `_get_model()`

## Steps

1. 构造 `embedder = Embedder(device="cpu")`。
2. 检查 `embedder._model`。
3. 可选：mock `FlagEmbedding.BGEM3FlagModel`，只在调用 `_get_model()` 后断言被调用。

## Expected Result

- 初始化后 `embedder._model is None`
- 构造 `Embedder` 本身不触发模型下载
