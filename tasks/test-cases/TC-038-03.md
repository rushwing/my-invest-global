## Objective

验证 parser 不只是返回 OCR 字段，还会调用 `analyze_chip` 派生 `ChipAnalysis.above_90_band` 等信号字段。

## Preconditions

- `parse_chip_screenshot` 可 import
- mock Claude API 返回 `current_price=131.60`、`range_90_upper=128.24`

## Steps

1. mock Claude API 返回完整 JSON，设置 `current_price > range_90_upper`。
2. 调用 `analysis = parse_chip_screenshot("tests/fixtures/chip_688143_20260519.png")`。
3. 断言：
   ```python
   assert analysis.above_90_band is True
   assert analysis.current_price == pytest.approx(131.60)
   assert analysis.range_90_upper == pytest.approx(128.24)
   ```

## Expected Result

- `above_90_band=True`
- 证明 parser 已通过 `analyze_chip` 生成完整 `ChipAnalysis`

## Notes

- 不需要 mock `analyze_chip`，除非实现需要验证调用次数
