## Objective

验证 CLI 单次采集入口可被 `uv run` 找到，且局部 source 失败不会导致进程失败。

## Preconditions

- `pyproject.toml` 注册 `refresh-macro-agent = "scripts.refresh_macro_agent:main"`
- 外部数据源均 mock，不访问真实网络

## Steps

1. 用 subprocess 执行 `uv run refresh-macro-agent --once --groups I,K`。
2. 在 mock 配置中让一个主源失败但备源成功。
3. 捕获进程退出码和 stderr。

## Expected Result

- 进程退出码为 0
- stderr/stdout 中没有未捕获 traceback
- fallback 后仍记录成功采集或跳过信息
