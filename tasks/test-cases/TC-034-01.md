## Objective

验证 `load_holdings_raw()` 返回未聚合的 8 条原始 holding，且可按 account 字段正确过滤出各账户的持仓数量。

## Preconditions

- `engine.portfolio.load_holdings_raw` 可 import
- `data/agent_input/cn/holdings.yaml` 存在且含 8 条 holding（华西 3 只、西南 5 只，其中 603986 各有一条）

## Steps

1. 调用 `raw = load_holdings_raw()`
2. 验证总数和各账户过滤结果：
   ```python
   assert len(raw) == 8
   huaxi = [h for h in raw if h.account == "华西证券"]
   xinan = [h for h in raw if h.account == "西南证券"]
   assert len(huaxi) == 3
   assert len(xinan) == 5
   ```
3. 验证 huaxi 包含 603986, 002384, 600150：
   ```python
   huaxi_codes = {h.code for h in huaxi}
   assert huaxi_codes == {"603986", "002384", "600150"}
   ```
4. 验证 xinan 包含 300308, 603986, 300394, 688820, 002957：
   ```python
   xinan_codes = {h.code for h in xinan}
   assert xinan_codes == {"300308", "603986", "300394", "688820", "002957"}
   ```

## Expected Result

- 总行数 8
- 华西 3 只（603986/002384/600150），西南 5 只（300308/603986/300394/688820/002957）
- 每行 account 字段均非空

## Notes

- 此测试不 mock YAML，直接使用真实的 holdings.yaml
- 与 TC-032-02 区别：TC-032-02 测试的是聚合逻辑，TC-034-01 测试的是 raw 加载的账户过滤
