## Objective

验证 `source_hash` 幂等规则符合 ADR-007 BP-10。

## Preconditions

- 使用 `MacroStorage(":memory:")`
- 同一公司同一季度的两次 CapEx records 拥有相同 `source_hash`
- 第二次记录的 `fetched_at` 晚于第一次

## Steps

1. 调用 `upsert_capex([record_v1])`。
2. 调用 `upsert_capex([record_v1_same_hash_later_fetched_at])`。
3. 查询该季度记录。

## Expected Result

- 第一次 upsert 返回 1
- 第二次 upsert 返回 0
- DB 中 `fetched_at` 仍等于第一次记录
- `capex_usd` 未变化
