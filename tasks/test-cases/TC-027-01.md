## Objective

验证 `FrozenMarketSnapshot` (Pydantic `frozen=True`) 在创建后对任何字段赋值都抛出 `ValidationError`。

## Preconditions

- `engine.agent.state.FrozenMarketSnapshot` 可正常 import
- `engine.schemas.HoldingRow`, `engine.schemas.PortfolioSummary` 已定义（mock 实例即可）

## Steps

1. 构造 `FrozenMarketSnapshot` 实例：
   ```python
   snap = FrozenMarketSnapshot(
       session_id="test-001",
       captured_at=datetime.utcnow(),
       holdings=(HoldingRow(...),),
       macro_state="green",
       portfolio_summary=PortfolioSummary(...),
       price_snapshot={"300308": 100.0},
       change_pct_snapshot={"300308": 2.5},
   )
   ```
2. 尝试 `snap.holdings = ()`

## Expected Result

- 第 2 步抛出 `pydantic.ValidationError`（或 `TypeError` / `FrozenInstanceError`）
- 异常消息指示字段不可变更

## Notes

- Pydantic v2 的 `model_config = {"frozen": True}` 行为等价于 `BaseModel(frozen=True)`
- `frozen=True` 阻止所有字段赋值，包括嵌套对象
