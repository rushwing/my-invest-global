## Objective

验证 `run_schema_init(conn)` 对 mock session 执行全部 `CONSTRAINTS + INDEXES`，且语句
均使用 `IF NOT EXISTS` 保持幂等。

## Preconditions

- mock `KGConnection.driver().session()` 返回 context manager session
- session 的 `run()` 已 mock

## Steps

1. 调用 `run_schema_init(conn)`。
2. 检查 `session.run` 的调用次数和入参。

## Expected Result

- `session.run` 调用次数等于 `len(CONSTRAINTS) + len(INDEXES)`，即 9 次
- 每条传入的 schema 语句都包含 `IF NOT EXISTS`
- 不需要真实 Neo4j 服务
