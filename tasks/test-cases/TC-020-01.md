## Objective

验证 `app.dashboard.main()` 在未解锁时可执行到锁定态 UI，不抛异常，并显示登录表单/锁定遮罩。

## Preconditions

- `app.dashboard.main` 可 import
- monkeypatch `app.auth.is_unlocked` 返回 `False`
- monkeypatch `render_sidebar(locked=True)` 与 `render_locked_mask`
- monkeypatch `streamlit.stop` 抛出可捕获的 sentinel exception

## Steps

1. 调用 `main()`。
2. 捕获 `st.stop()` sentinel。
3. 断言：
   ```python
   render_sidebar.assert_called_once_with(locked=True)
   render_locked_mask.assert_called_once()
   ```

## Expected Result

- 启动路径无 Python 异常
- 未解锁时显示登录/锁定 UI
- 不渲染 4 个 Tab 内容

## Notes

- 单元测试不启动真实 Streamlit server；可另有 smoke test 覆盖 `streamlit run`
