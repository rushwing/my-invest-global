## Objective

验证新闻时间线在输入超过 8 条新闻时，仅渲染最近 8 条，且按发布时间倒序。

## Preconditions

- `app.components.tab3_stock.render_news_timeline` 或等价函数可 import
- 可 monkeypatch `streamlit.markdown` 捕获渲染顺序
- 准备 10 条 `published_at` 不同的新闻 DataFrame

## Steps

1. 将新闻 DataFrame 打乱顺序传入 `render_news_timeline(news_df)`。
2. 捕获每次渲染的标题。
3. 断言：
   ```python
   assert len(rendered_titles) == 8
   assert rendered_titles == expected_latest_8_titles_desc
   ```

## Expected Result

- 最多渲染 8 条新闻
- 渲染顺序为 `published_at` 倒序

## Notes

- 不要求真实新闻源
