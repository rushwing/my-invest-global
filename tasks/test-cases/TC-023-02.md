## Objective

通过静态检查验证 `Institution` 与 `HOLDS` 写入语句使用 `MERGE` 保持幂等。

## Preconditions

- 可 import `engine.kg.institution_loader`

## Steps

1. 读取 `_MERGE_INSTITUTION` 和 `_MERGE_HOLDS` 两个 Cypher 字符串。
2. 移除允许的 `ON CREATE SET` 子句文本后检查是否还有裸 `CREATE`。

## Expected Result

- 两个 Cypher 字符串都包含 `MERGE`
- 两个 Cypher 字符串都不包含裸 `CREATE `
- 允许出现 `ON CREATE SET`
