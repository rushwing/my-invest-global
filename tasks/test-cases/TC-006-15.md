## Objective

验证 `fetch_cls_news()` 改用 `_make_period_record()` 后，
返回记录包含 `value_unit`, `vintage_as_of`, `realtime_start` 等字段，
消除与 REQ-007 upsert schema 的不兼容。

## Steps

1. Mock `src._ak_call` 返回：
   ```python
   pd.DataFrame([{"title": "美联储降息", "content": "..."}])
   ```
2. 调用 `src.fetch_cls_news(limit=1)`

## Expected Result

- 返回 1 条记录
- `rec["indicator_id"]` == `"CLS_NEWS"`
- `rec["value"]` is None
- `rec` 包含 `value_unit`, `vintage_as_of`, `realtime_start` 字段
  （来自 `_make_period_record()` 的标准输出）
