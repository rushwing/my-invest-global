## Objective

验证数据不可用时 `fetch_chip_summary` 不静默返回 `None` 或空 model，而是抛出 `ChipDataUnavailable`。

## Preconditions

- `fetch_chip_summary` 和 `ChipDataUnavailable` 可 import
- mock `akshare.stock_cyq_em` 返回空 `pd.DataFrame()`

## Steps

1. patch `akshare.stock_cyq_em` 返回空 DataFrame。
2. 调用：
   ```python
   with pytest.raises(ChipDataUnavailable) as exc:
       fetch_chip_summary("688143")
   ```
3. 断言异常信息包含股票代码：
   ```python
   assert "688143" in str(exc.value)
   ```

## Expected Result

- 抛出 `ChipDataUnavailable`
- 不返回 `None`
- 不返回字段缺失的 `ChipSummary`

## Notes

- 此 TC 覆盖 akshare 数据不可用的降级行为
