## Objective

验证坏数据 `avg_cost <= 0` 不会导致除零或无意义信号，而是抛出 `ValueError("Invalid avg_cost")`。

## Preconditions

- `analyze_chip` 可 import
- 准备 `avg_cost=0.0` 的 mock `ChipSummary`

## Steps

1. 构造 `summary.avg_cost = 0.0`。
2. 调用：
   ```python
   with pytest.raises(ValueError, match="Invalid avg_cost"):
       analyze_chip("688143", 131.60, summary)
   ```

## Expected Result

- 抛出 `ValueError`
- 错误信息包含 `Invalid avg_cost`

## Notes

- 可额外覆盖负数 avg_cost
