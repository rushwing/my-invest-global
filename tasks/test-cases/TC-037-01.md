## Objective

验证 `analyze_chip("688143", 131.60, mock_summary_688143)` 返回 `ChipAnalysis`，并正确计算突破 90% 筹码上界、成本偏离率和集中度等级。

## Preconditions

- `engine.agent.chip_analysis.analyze_chip` / `ChipAnalysis` 可 import
- `engine.agent.chip_fetcher.ChipSummary` 可 import
- mock summary: `avg_cost=99.96`，`range_90_upper=128.24`，`range_90_lower=63.28`，`concentration=33.92`

## Steps

1. 构造 `mock_summary_688143`。
2. 调用 `analysis = analyze_chip("688143", 131.60, mock_summary_688143)`。
3. 断言：
   ```python
   assert isinstance(analysis, ChipAnalysis)
   assert analysis.above_90_band is True
   assert analysis.below_90_band is False
   assert analysis.cost_deviation_pct == pytest.approx(31.7, abs=0.1)
   assert analysis.chip_lock_level.value == "low"
   assert len(analysis.signal_summary) <= 80
   ```

## Expected Result

- 返回 `ChipAnalysis`
- `above_90_band=True`
- `cost_deviation_pct` 约为 `+31.7`
- `chip_lock_level=LOW`

## Notes

- 纯函数测试，不访问数据库或网络
