## Objective

验证 Gauge 对 `abs(deviation)=11%` 使用红色 `#E84040`，并在触发器表中产生红色背景的 triggered 行。

## Preconditions

- Gauge 颜色 helper 或渲染函数可 import
- `compute_trigger_status()` 可 import
- 可 monkeypatch 表格渲染函数捕获 status/style

## Steps

1. 构造弹性仓位比目标偏离 11% 的 holdings DataFrame。
2. 调用 Gauge 颜色计算与 `compute_trigger_status()`。
3. 渲染 RebalanceTriggers 表或检查 style helper。
4. 断言：
   ```python
   assert gauge_color == "#E84040"
   assert "triggered" in set(triggers["status"])
   assert triggered_row_style_contains_red_background
   ```

## Expected Result

- Gauge 为 bear 红色
- 至少一行触发器状态为 `triggered`
- `triggered` 行使用红色背景样式

## Notes

- 不要求真实市场数据
