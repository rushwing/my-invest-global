## Objective

验证 MCP 工具不是孤立函数，而是注册到 `engine.mcp.server.mcp`。

## Preconditions

- `engine/mcp/server.py` 定义 `mcp = FastMCP("my-invest-global")`
- 不启动真实 MCP 服务进程

## Steps

1. 执行 `from engine.mcp.macro_tools import get_macro_state`。
2. 执行 `from engine.mcp.server import mcp`。
3. 调用 `mcp.get_tool("get_macro_state")` 或等价的 FastMCP 工具查询 API。

## Expected Result

- 导入不抛 `ImportError`
- `get_macro_state` 可调用
- 共享 `mcp` 实例中存在名为 `get_macro_state` 的工具
