## Objective

验证 `render_chip_panel` 在 `profitable_pct=0.9973` 时，指标卡展示 `"99.7%"`，并设置非空 delta 警告。

## Preconditions

- `render_chip_panel` 可 import
- mock `streamlit` 指标调用
- mock analysis: `profitable_pct=0.9973`，`cost_deviation_pct` 有明确值

## Steps

1. monkeypatch `st.metric` 为 mock。
2. 调用 `render_chip_panel(...)`。
3. 检查 `st.metric` 调用参数：
   ```python
   calls = st.metric.call_args_list
   assert any("99.7%" in str(call) for call in calls)
   assert any("delta" in call.kwargs and call.kwargs["delta"] for call in calls)
   ```

## Expected Result

- 收盘获利指标显示 `99.7%`
- 至少一个 metric 的 delta 非空，用于提示获利盘压力或偏离率

## Notes

- 不要求固定中文 delta 文案，只要求有警示信息
