## Objective

验证 `compute_indicators(load_ohlcv("300308", days=180))` 生成的 `rsi14` 列所有非空值均在 `[0, 100]`，且 RSI warm-up NaN 被允许。

## Preconditions

- `app.data_loader.compute_indicators` 可 import
- 准备 180 行按 `date` 升序排列的 OHLCV DataFrame，列含 `open/high/low/close/volume`

## Steps

1. 调用 `df = compute_indicators(ohlcv_df)`。
2. 提取 `rsi = df["rsi14"]`。
3. 断言：
   ```python
   assert rsi.notna().sum() > 0
   assert rsi.isna().sum() <= 13
   assert rsi.dropna().between(0, 100).all()
   ```

## Expected Result

- `rsi14` 列存在
- 所有非空 RSI 值位于 `[0, 100]`
- warm-up NaN 不被视为越界

## Notes

- 纯函数测试，不访问 DuckDB 或 AkShare
