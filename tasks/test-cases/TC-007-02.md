## Objective

验证 `MacroStorage.upsert_indicators()` 不丢失 FRED vintage 字段，并且
`get_latest_indicator(indicator_id, source=None)` 按 `fetched_at DESC` 返回最新一行。

## Preconditions

- 使用 `MacroStorage(db_path=":memory:")`
- 两条记录使用同一 `indicator_id="CPIAUCSL"`，但不同 `period_date`

## Fixture

记录 A：

```python
{
    "indicator_id": "CPIAUCSL",
    "group_code": "I",
    "period_date": date(2026, 3, 1),
    "market_tz": "America/New_York",
    "value": 3.4,
    "value_unit": "percent_yoy",
    "value_raw": "3.4",
    "source": "fred",
    "realtime_start": date(2026, 4, 1),
    "vintage_as_of": date(2026, 4, 11),
    "utc_ts": datetime(2026, 4, 11, 12, 0, tzinfo=UTC),
    "fetched_at": datetime(2026, 4, 11, 12, 0, tzinfo=UTC),
}
```

记录 B 的 `period_date=date(2026, 4, 1)`、`value=3.5`、
`realtime_start=date(2026, 5, 1)`、`vintage_as_of=date(2026, 5, 12)`、
`fetched_at=datetime(2026, 5, 12, 12, 0, tzinfo=UTC)`。

## Steps

1. 构造 `storage = MacroStorage(":memory:")`
2. 调用 `storage.upsert_indicators([record_a, record_b])`
3. 调用 `latest = storage.get_latest_indicator("CPIAUCSL")`
4. 调用 `latest_fred = storage.get_latest_indicator("CPIAUCSL", source="fred")`

## Expected Result

- `latest["period_date"] == date(2026, 4, 1)`
- `latest["realtime_start"] == date(2026, 5, 1)`
- `latest["vintage_as_of"] == date(2026, 5, 12)`
- `latest["source"] == "fred"`
- `latest_fred` 与 `latest` 指向同一条最新记录
