## Objective

验证 `ingest_pdf(meta, chunks, db, mock_embedder)` 会批量向 LanceDB 写入 chunk，并返回
写入行数。

## Preconditions

- 使用 `lancedb.connect(":memory:")`
- 构造 3 个 `ParsedChunk`
- mock embedder 的 `embed(texts)` 返回 shape `(3, 1024)` 的 numpy array

## Steps

1. 构造 `DocMeta` 和 3 个 chunks。
2. 调用 `ingest_pdf(meta, chunks, db, mock_embedder)`。
3. 打开 `doc_chunks` 表并查询行数。

## Expected Result

- `ingest_pdf()` 返回 `3`
- LanceDB 表中行数为 3
- 写入的 `chunk_total` 均为 3
