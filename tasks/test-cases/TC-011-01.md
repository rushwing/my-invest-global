- mock `get_capex_quarters()` 使 Big-4 QoQ = +8%

## Steps

1. 构造 `MacroRegime(storage)`。
2. 调用 `compute(as_of=date(2026, 5, 15))`。

## Expected Result

- `capex_state == "green"`
- `yield_curve_state == "normal"`
- `risk_state == "risk_on"`
- `composite_state == "green"`
