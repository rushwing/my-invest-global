## Objective

验证 `fetch_ohlcv()` 在 yfinance 返回 tz-naive DatetimeIndex（日线常见）时
能正确提取 `period_date`，而非被 broad except 吞掉后返回空列表。

## Preconditions

- yfinance.download mock 返回含 tz-naive `pd.Timestamp` 索引的 DataFrame

## Steps

1. 构造 `YahooGlobalSource(rate_limiter)`
2. Mock `yfinance.download` 返回：
   ```python
   pd.DataFrame(
       {"Close": [150.0]},
       index=[pd.Timestamp("2026-05-15")]  # tz-naive
   )
   ```
3. 调用 `src.fetch_ohlcv(["MSFT"])`

## Expected Result

- 返回 1 条记录（非空列表）
- `rec["indicator_id"]` == `"MSFT"`
- `rec["period_date"]` == `date(2026, 5, 15)`
- `rec["value"]` == `150.0`
