## Objective

验证 `AlphaVantageSource.fetch_news_sentiment()` 在预算计数器 ≥ 22 时
立即 raise `BudgetExhaustedError`，不发送任何 HTTP 请求。

## Steps

1. 构造 `AlphaVantageSource`，注入 `get_budget=lambda: 22`，
   `inc_budget=lambda: None`，`api_key="test"`
2. 用 mock 替换 `_get`（用于确认未被调用）
3. `with pytest.raises(BudgetExhaustedError):`
   `    src.fetch_news_sentiment()`
4. 断言 mock `_get` 的 `call_count == 0`

## Expected Result

- `BudgetExhaustedError` 在 HTTP 请求前抛出
- `_get` 未被调用（预算检查在 HTTP 层之前）
