## Objective

验证 ScarcityMatrix chip 对 `composite_score=72` 使用 bull 绿色 `#00C47A`。

## Preconditions

- `app.components.tab4_signals.chip_color` 或等价 helper 可 import

## Steps

1. 调用 `color = chip_color(72)`。
2. 断言：
   ```python
   assert color == "#00C47A"
   ```

## Expected Result

- 分数大于等于 70 的 chip 使用绿色

## Notes

- 覆盖高分阈值
