## Objective

验证 Claude API 返回非 JSON 字符串时，parser 抛出 `ScreenshotParseError`，错误原因包含 parse 或 JSON。

## Preconditions

- `parse_chip_screenshot` 和 `ScreenshotParseError` 可 import
- mock Claude API 返回 `"not json"` 或带解释文本的非 JSON 内容

## Steps

1. mock Claude API 返回非法 JSON。
2. 调用：
   ```python
   with pytest.raises(ScreenshotParseError) as exc:
       parse_chip_screenshot("tests/fixtures/chip_688143_20260519.png")
   assert "parse" in str(exc.value).lower() or "json" in str(exc.value).lower()
   ```

## Expected Result

- 抛出 `ScreenshotParseError`
- 错误原因可定位为 JSON/parse 问题

## Notes

- 此 TC 不覆盖网络/API 调用失败；实现可额外测试 `API call failed`
