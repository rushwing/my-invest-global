## Objective

验证 `portfolio_analyzer(state)` 从 `FrozenMarketSnapshot` 提取每个持仓的上下文字符串，写入 `kg_subgraph`。

## Preconditions

- `engine.agent.nodes.portfolio_analyzer` 可 import
- 构造 `AnalysisState`，其中 `snapshot.holdings` 含 2 个 `HoldingRow`（code="300308"/"002230"）

## Steps

1. 构造 snapshot 含 2 个 `HoldingRow`，每只股票有 `cost_price`, `current_price`, `market_value`, `sector` 字段
2. 构造 `AnalysisState`，其中 `snapshot` 为上一步的 snapshot
3. 调用 `result = portfolio_analyzer(state)`

## Expected Result

- `result["kg_subgraph"]` 为 `dict`，包含 2 个 key：`"300308"` 和 `"002230"`
- 每个 key 对应的 `list` 非空（至少 4 个字符串）
- 每个 list 中包含 `持仓成本`、`当前价格`、`市值`、`板块:` 前缀的字符串
- `result["errors"]` == `[]`
- `result["rag_chunks"]` == `{}`

## Notes

- `portfolio_analyzer` 不依赖外部调用，是纯数据提取函数
- `kg_subgraph` 在此节点中填充 context 字符串，后续 `kg_retrieval` 节点追加 KG 查询结果
