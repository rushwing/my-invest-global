## Objective

验证 FundamentalCard 渲染 Tier-2 股票时，Tier 徽章颜色为 `#F5A623`。

## Preconditions

- `app.components.tab3_stock.render_fundamental_card` 或等价函数可 import
- 可 monkeypatch `streamlit.markdown` 捕获 HTML
- 准备 stock metadata：`rank=2`, `code="300308"`, `name="中际旭创"`

## Steps

1. monkeypatch `st.markdown`，记录传入的 HTML 字符串。
2. 调用 `render_fundamental_card(meta, latest_price=100.0)`。
3. 断言捕获的 HTML 中包含：
   ```python
   assert "Tier-2" in html
   assert "#F5A623" in html
   ```

## Expected Result

- Tier-2 徽章可见
- 徽章颜色为 `#F5A623`

## Notes

- 若实现拆出 `tier_color(rank)`，可直接测试该 helper
