## Objective

验证某股票无 fund_flow 数据时，mini strip 显示 5 个灰色占位格子，并且渲染函数不抛异常。

## Preconditions

- `app.components.tab4_signals.render_flow_strip` 或等价函数可 import
- 准备空 DataFrame，列含 `code/trade_date/main_net_inflow`

## Steps

1. 调用 `html = render_flow_strip(empty_flow_df, code="300308")`。
2. 断言：
   ```python
   assert html.count("flow-cell") == 5
   assert "gray" in html or "#9AA0AC" in html
   ```

## Expected Result

- 空数据不抛异常
- 显示 5 个灰色占位格子

## Notes

- 该行为支撑无表/缺数据降级
