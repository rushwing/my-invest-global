## Objective

验证 `FREDSource.fetch_series()` 在正常响应下返回 list[dict]，且每条记录含
`realtime_start` 和 `vintage_as_of` 字段。

## Preconditions

- `FRED_API_KEY` 环境变量已设置（值任意，mock 不校验）
- HTTP GET 已 mock（不发真实请求）

## Steps

1. 用 `unittest.mock.patch` 替换 `FREDSource._get`，返回 FRED 标准格式 JSON：
   ```json
   {
     "observations": [
       {"date": "2026-04-01", "value": "3.5", "realtime_start": "2026-04-11"}
     ]
   }
   ```
2. 构造 `FREDSource(rate_limiter)`，调用
   `fetch_series("CPIAUCSL", start_date="2026-01-01", realtime_start="2026-04-01")`

## Expected Result

- 返回 `list[dict]`，长度 ≥ 1
- `result[0]["realtime_start"]` == `"2026-04-01"`（或 `date(2026, 4, 1)`）
- `result[0]["vintage_as_of"]` 为今日日期（`date.today()`）
- `result[0]["indicator_id"]` == `"CPIAUCSL"`
- `result[0]["value"]` == `3.5`（float）
