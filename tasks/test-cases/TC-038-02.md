## Objective

验证 Claude 返回无筹码字段或关键字段为 `null` 时，parser 抛出 `ScreenshotParseError`，不返回部分填充 model。

## Preconditions

- `parse_chip_screenshot` 和 `ScreenshotParseError` 可 import
- mock Claude API 返回 `avg_cost=null`、`profitable_pct=null` 等缺失关键字段的 JSON

## Steps

1. mock Claude API 返回：
   ```json
   {"code":"688143","avg_cost":null,"profitable_pct":null}
   ```
2. 调用：
   ```python
   with pytest.raises(ScreenshotParseError):
       parse_chip_screenshot("tests/fixtures/chip_688143_20260519.png")
   ```

## Expected Result

- 抛出 `ScreenshotParseError`
- 不返回 `ChipAnalysis`

## Notes

- 用于覆盖不适用截图或 OCR 失败路径
