## Objective

验证 `fetch_chip_summary("688143")` 在 mock `ak.stock_cyq_em` 返回标准 DataFrame 时，能生成字段完整的 `ChipSummary`，并归一化比例字段。

## Preconditions

- `engine.agent.chip_fetcher.fetch_chip_summary` 可 import
- `engine.agent.chip_fetcher.ChipSummary` / `ChipBar` 可 import
- 使用 `unittest.mock.patch` mock `akshare.stock_cyq_em`，不访问真实网络

## Steps

1. 构造 mock DataFrame，至少包含 `日期`、`获利比例`、`平均成本`、`90成本-低`、`90成本-高`、`70成本-低`、`70成本-高`、`集中度`，以及至少两档逐档筹码数据。
2. patch `akshare.stock_cyq_em` 返回该 DataFrame。
3. 调用 `summary = fetch_chip_summary("688143")`。
4. 断言：
   ```python
   assert isinstance(summary, ChipSummary)
   assert summary.code == "688143"
   assert summary.avg_cost > 0
   assert 0 < summary.profitable_pct <= 1
   assert 0 <= summary.concentration <= 100
   assert summary.range_90_lower < summary.range_90_upper
   assert len(summary.bars) > 0
   assert all(0 <= b.chip_ratio <= 1 for b in summary.bars)
   ```

## Expected Result

- 返回 `ChipSummary`
- `profitable_pct` 和 `ChipBar.chip_ratio` 均为 0-1 小数
- `bars` 非空，且每个 bar 有合法价格区间

## Notes

- 此 TC 只验证字段映射和基础校验，不依赖真实 akshare
- mock 列名按 REQ-036 的映射表构造
