## Objective

验证 `load_market_breadth(days=20)` 从 DuckDB `stock_quotes` 聚合每日上涨家数比例，返回 `up_ratio` 在 `[0,1]` 且行数不超过 20。

## Preconditions

- `app.data_loader.load_market_breadth` 可 import
- 使用临时 DuckDB 创建 `stock_quotes(quote_time, pct_change)`
- 插入最近 20 天内的涨跌 mock 数据

## Steps

1. monkeypatch `DB_PATH` 指向临时 DuckDB。
2. 调用 `df = load_market_breadth(days=20)`。
3. 断言：
   ```python
   assert list(df.columns) == ["date", "up_ratio"]
   assert len(df) <= 20
   assert df["up_ratio"].between(0, 1).all()
   ```

## Expected Result

- 返回每日市场宽度 DataFrame
- `up_ratio` 值均在 `[0,1]`
- 参数化日期过滤 SQL 可执行

## Notes

- 此 TC 防止 `INTERVAL ? DAY` 这类不可解析 SQL 回归
