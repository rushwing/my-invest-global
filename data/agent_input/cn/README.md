# CN AIDC Agent Input

按截图中的 9 大领域拆分自 `data_china_aidc.md`：

1. 光通信
2. 内存/半导体
3. 算力/芯片
4. PCB
5. 上游材料
6. AI服务器
7. 设备/封测
8. 散热/液冷
9. 电源系统

`stocks.yaml` 只维护股票名称、代码、交易所/归属板、领域、细分板块、来源文件等确定性元数据。行情、财报、评级、紧缺度、产品线营收/净利份额比例、估值/业绩弹性、戴维斯双击观察等随时间变化或需要判断的信息保留在各分类 Markdown 中。

刷新方式：

```bash
uv run python scripts/refresh_aidc_data.py
```

脚本会从 `stocks.yaml` 读取股票池，刷新股价、当日涨跌幅、成交量、成交额、总市值、动态 PE、近一月/三月/半年/一年涨跌幅，并从东方财富 F10 主营构成获取产品线营收/利润份额，再重写主表、各分类 Markdown 和 HTML。

`_out_of_scope.md` 保留源表里不属于截图 9 大领域的条目，防止拆分时丢失溯源信息。
