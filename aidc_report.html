方法论框架 — 时寒冰四步投资法

① 宏观先行 — 主权级不可逆投入

IEA预测2026年AIDC耗电945 TWh；四大超大云厂商2026年资本支出合计 $3,150亿（+48% YoY）。AI算力建设具主权竞争属性，资金流向不可逆是本轮周期最核心判断。

② 稀缺节点 — 投资优先级矩阵

优先级分 = 稀缺度 × (1/可替代性) × 需求弹性。当前优先级：光通信/内存/国产算力 → PCB/上游材料 → 算力租赁/服务器 → 设备封测/散热/电源。

③ 中国特色 — 国产替代三波浪

第一波（已兑现）：光模块/基础服务器。第二波（当前）：CPO/高速PCB/存储接口/算力租赁。第三波（2027-28）：先进封装、SST供配电、金刚石热管理。

④ 估值纪律 — 金字塔仓位框架

核心仓40%（Tier-1，集中2-3只）；卫星仓35%（Tier-2，分散4-6只）；机动仓25%（Tier-3弹性品种）。卖出触发：超大云厂商资本支出季度下调>10%，板块PE>历史峰值1.5×。

中国 A 股 · AIDC 产业链全景图
中国 AIDC 板块轮动顺序：光通信 → 内存/半导体 → 算力/芯片 → PCB → 上游材料 → AI服务器 → 半导体设备/封测 → 散热/液冷 → 电源系统

1. 光通信

中际旭创 / 新易盛 / 天孚通信 / 剑桥科技。800G/1.6T、CPO与光互联仍是最强主线。

2. 内存/半导体

澜起科技、江波龙、佰维存储。RCD/DDR5、企业级SSD与存储周期共振。

3. 算力/芯片

寒武纪、海光信息。国产训练芯片与高端处理器是主权算力核心。

4. PCB

深南电路、沪电股份、胜宏科技。高多层、高速背板、AI服务器板卡主线。

5. 上游材料

生益科技、东材科技、圣泉集团、福晶科技。CCL、电子树脂、光学晶体作为扩散线。

6. AI服务器

浪潮信息、中科曙光、利通电子、协创数据。整机与算力租赁分开跟踪。

7. 设备/封测

北方华创、中微公司、长电科技、通富微电。设备是前端，封测对应先进封装放量。

8. 散热/液冷

英维克、申菱环境、高澜股份、沃尔德。液冷为主线，金刚石散热先按观察处理。

9. 电源系统

金盘科技、科华数据、科士达、动力源。补入SST、HVDC、数据中心供配电设备。
